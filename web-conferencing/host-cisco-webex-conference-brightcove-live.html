    Veranstalten einer Cisco Webex Konferenz zu Brightcove Live

    In diesem Thema erfahren Sie, wie Sie eine Cisco Webex-Konferenz mit Brightcove Live streamen.

    Einführung

    Cisco Webex ist eine Videokonferenzanwendung, mit der Sie sich online mit anderen Personen treffen können. Um eine größere Zielgruppe zu erreichen, kannst du Brightcove Live nutzen, um dein Meeting für andere zu streamen.

    Die Schritte

    Befolgen Sie diese Schritte, um ein Meeting für Brightcove Live zu veranstalten:

    1. Melden Sie sich als Site-Adminbereich bei Ihrer Cisco Webex-Plat

    2. Aktivieren Sie entweder den Site-Administration oder den Control Hub , um den Live-Stream-Zugriff auf andere Streaming-Dienstezu ermöglichen.

    3. Aktivieren Sie entweder die Site-Administration oder den Control Hub , indem Sie Live-Streaming für einzelne Benutzer verwenden.

      Gehen Sie zu Benutzer und wählen Sie ein zu aktualisierendes Benutzerkonto aus. Klicken Sie für den Benutzer auf den Abschnitt Erweiterte Benutzereinstellungen und Tracking-Codes und aktivieren Sie das Kontrollkästchen Live-Streaming aktivieren.

    4. Melden Sie sich bei Brightcoves Video Cloud Studioan.

    5. Navigieren Sie zum Live-Modul und erstellen Sie ein Brightcove Live Event. Einzelheiten finden Sie im Dokument Erstellen und Verwalten von Live-Events mit dem Live Module .

      Wenn Sie die Sendung nach Abschluss speichern möchten, aktivieren Sie das Kontrollkästchen Ereignis nach Abschluss in Video konvertieren !

    6. Wenn Sie dieses Ereignis erstellen, erhalten Sie die folgenden Informationen:

      • Endpunkt (RTMP-URL) - Streaming-URL
      • Streamname - Streaming-Schlüssel, bei dem der Wert immer am Lebenist
    7. Veröffentlichen und betten Sie den Spieler für dieses Live-Event in Ihre Website oder Intranetseite ein. Kopiere die Seiten-URL in deine Zwischenablage.

    8. Kehren Sie zur Cisco Webex-Anwendung zurück. Beginnen oder planen Sie Ihr Webex-Meeting. Wählen Sie dazu das vertikale 3-Punkt-Symbol aus und wählen Sie Live-Streaming starten.

    9. Geben Sie im Dialogfeld Weitere Informationen zum Streaming-Dienst Folgendes ein:

      • Streaming-Dienst: Benutzen Brightcove
      • Ziel-Streamlink: Verwenden Sie den Endpoint (RTMP-URL) -Wert von Brightcove Live.
      • Schlüssel für Zielstreams: Verwenden alive (Es ist immer dieser Wert)

    10. Das ist es. Du bist live!
