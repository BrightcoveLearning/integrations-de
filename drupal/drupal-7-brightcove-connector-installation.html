Drupal 7-Brightcove-Anschluss: Installation

Einführung

Mit dem Drupal-Brightcove-Connector können Sie Brightcove Video Cloud-Videos und -Player in Drupal verwalten und Videos einfach in Drupal-Seiten einbetten. In diesem Artikel wird speziell auf die Installation des Drupal 7-Plug-Ins eingegangen. Für Drupal 8 siehe bitte die Installationsanleitung für Drupal 8.

Die Drupal-Installation ist ein komplizierter Prozess, der sehr leicht schief geht. Stellen Sie sicher, dass Ihre Inhalte gesichert sind, bevor Sie fortfahren.

Installationsschritte

Stellen Sie für optimale Ergebnisse sicher, dass Sie Composer bereits installiert haben. Hier finden Sie Anweisungen, wie Hier.

  1. Herunterladen und installieren der PHP-API-Wrapper. Ziehen nach sites/all/libraries/PHP-API-Wrapper
  2. Navigieren Sie zu sites/all/libraries/PHP-API-Wrapper in Ihrer Konsole. Führen Sie im Ordner Folgendes aus composer install.
  3. Laden Sie jetzt die Brightcove-Modul. Sie müssen auch einige andere Module für Abhängigkeiten laden:
    • Chaos Werkzeuge
    • Entitäts-API
    • Bibliotheks-API
    • Datum
    • Dateieinheit
    • Medien
    • Ansichten
  4. Verschieben Sie alle Module in die modules Mappe.
  5. Gehen Sie zu Ihrer Drupal-Site. Navigieren Sie zur Seite Module.
  6. Aktivieren Sie die folgenden Module: ctools, Entitäts-API, Bibliotheken, Datum, Datums-API, Datums-Popup, Medien, Medien-Internetquellen, Datei-Entität und Ansichten. Diese sollten alle dank Schritt 3 verfügbar sein.
  7. Stellen Sie unbedingt Datum und Uhrzeit ein! Ihr Brightcove-Modul funktioniert nicht, wenn Sie dies nicht tun.

Sobald Sie fertig sind, können Sie zu wechseln Aufbau.
