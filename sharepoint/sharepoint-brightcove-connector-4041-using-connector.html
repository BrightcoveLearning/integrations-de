    SharePoint-BrightCove Anschluss 4.0/4.1: Verwenden des Connectors

    Videos hinzufügen und bearbeiten

    Der SharePoint Connector ermöglicht es einem Inhaltsautor, Videos von seinem lokalen Computer hochzuladen und Metadaten für die Videos zu bearbeiten.

    Videos hinzufügen

    Der Abschnitt „Videos hinzufügen“ des Connectors wird verwendet, um neue Videos in Video Cloud hochzuladen. Wenn ein Benutzer auf „Videos hinzufügen“ zugreift (wenn er dazu berechtigt ist), werden ihm die folgenden Felder angezeigt, die es ihm ermöglichen, die Metadaten einzugeben, die an Video Cloud gesendet werden:

    • Konto auswählen - Diese Dropdown-Liste wird mit allen Konten gefüllt, zu denen der Benutzer berechtigt ist, Videos hinzuzufügen. Dies wird über die Zuweisung von Kontogruppen aktiviert. Der Benutzer muss ein Konto auswählen, um das Hochladen zu ermöglichen.
    • Videodatei - Ziehen Sie Videodateien in das Feld oder klicken Sie auf die Schaltfläche „Durchsuchen“, um den lokalen Dateisystem-Explorer zu öffnen und die Videodatei für den Upload auszuwählen. Dies kann mehrfach wiederholt werden, ohne die Bildschirme wechseln zu müssen. Beachten Sie, dass dieser Schritt nach Folder und Brightcove Tags ausgeführt werden muss, wenn Sie sie von diesem Bildschirm aus festlegen möchten. Sie können auch später bearbeiten.
    • Ordner - Videos können zu organisatorischen Zwecken in bestimmten Ordnern gespeichert werden.
    • Brightcove tags - Daten-Tags, die zusätzlich zum Speichern in SharePoint in den Video Cloud-Tag-Informationen gespeichert werden. Diese unterscheiden sich von SharePoint-Metadaten-Tags, die einige Clients dieser Schnittstelle hinzufügen möchten und die nur in SharePoint gespeichert (nicht in Video Cloud übertragen) gespeichert werden.

    Wenn die Verarbeitung des Videos abgeschlossen ist, wird ein Link „Video bearbeiten“ angezeigt, über den der Benutzer Metadaten hinzufügen kann.

    Videos bearbeiten

    Der Bildschirm „Videos bearbeiten“ kann sowohl über den Link „Video bearbeiten“ nach dem Hochladen als auch durch Navigieren zu „Videos verwalten“ und auf das gewünschte Video geklickt werden.

    Der Benutzer kann auch nach Videos suchen, indem er Suchbegriffe in das Feld „Verfügbare Videos“ eingibt und die Daten sortieren soll. Wenn Sie auf den Namen des Videos oder den Link „Video bearbeiten“ klicken, wird der Benutzer zur Seite Video bearbeiten weitergeleitet.

    Das Brightcove Video Cloud-Konto, in das das Video hochgeladen wird, kann nicht geändert werden, aber die meisten anderen Informationen, einschließlich Videonamen und Beschreibungen und aller benutzerdefinierten Felder, die Ihr Konto enthält. Der Name wird aus dem Dateinamen automatisch generiert, kann jedoch geändert werden.

    Ein Poster und ein Thumbnail werden ebenfalls generiert, können aber auch hier geändert werden. Es gibt zwei Möglichkeiten, die Bilder zu ändern. Sie können einen neuen Screencap aus dem Video aufnehmen, indem Sie den Aufnahmeschaltknopf in der oberen rechten Ecke des Videos verwenden, oder Sie können einfach neue Bilder hochladen.

    Hinzufügen von Textspuren

    Textspuren können jedem Video entweder beim ersten Upload oder bei nachfolgenden Updates hinzugefügt werden. Mit einem Video können eine oder mehrere Textspuren verknüpft sein. Der Benutzer kann die Art der Textspur auswählen, die dem Video hinzugefügt werden soll (Datei-Upload, URL-Upload oder Remote-URL). Wenn „Keine“ ausgewählt ist, wird in dem Abschnitt keines der Felder angezeigt.

    WICHTIG: Ein einzelnes Video wird KEINE Mischung aus Datei-Uploads, URL-Uploads oder Remote-URLs unterstützen. Mit jedem Video kann nur eine Art von Textspur verknüpft werden.

    Datei-Upload

    Wenn „Datei-Upload“ für Text Track Files ausgewählt ist, werden die folgenden Felder angezeigt:

    1. Typ der Textspur auswählen - Ermöglicht die Auswahl von Datei-Upload, URL-Upload oder Remote-URL.
    2. File Chooser - Auswahl der Textspurdatei.
    3. Textspureigenschaften - Legen Sie Eigenschaften fest, die mit der Textspur verknüpft sind. Wenn Sie auf „Hinzufügen“ klicken, wird die Textspur beim nächsten Upload/Update zu den Videodaten hinzugefügt.
      1. Sprache - Der zweibuchstabige Code (gültiges Sprach-Tag BCP 47) für die Sprache der Textspur, zum Beispiel „en“ für Englisch.
      2. Label - Das Label für den Track, das für den Benutzer sichtbar ist, z. B. in einem Menü, das die verschiedenen für Untertitel verfügbaren Sprachen auflistet.
      3. Art - Einer der fünf aufgelisteten unterstützten Track-Typen
    4. Liste der hinzugefügten/in der Warteschlange stehenden Textspuren - Zeigt die Liste der vorhandenen Textspuren und neuen Textspuren an, die beim nächsten Speichern in die Warteschlange gestellt werden. Eine Textspur kann gelöscht werden, indem Sie auf den Link „Entfernen“ klicken und das Video speichern.

    URL hochladen, Remote-URL-Hochladen

    Wenn Sie eine dieser Optionen auswählen, wird der Upload „Datei-Upload“ durch ein Textfeld ersetzt, in das der vollständig qualifizierte URL-Pfad zur Textspurdatei eingegeben werden kann.

    • URL Upload - Geben Sie die öffentliche URL ein, in der sich die Textspurdatei befindet, und klicken Sie dann auf Hochladen. Die Datei wird in Video Cloud hochgeladen.
    • Remote-URL - Geben Sie die öffentliche URL ein, in der sich die Textspurdatei befindet. Die URL darf maximal 249 Zeichen lang sein. Untertitel werden von der externen URL geladen.
    1. Typ der Textspur auswählen - Ermöglicht die Auswahl von Datei-Upload, URL-Upload oder Remote-URL.
    2. URL hinzufügen - Feld, um den vollqualifizierten URL-Pfad zur Texttrack-Datei einzugeben.
    3. Eigenschaften der Textspur - Ermöglicht das Festlegen der mit der Textspur verbundenen Eigenschaften. Wenn Sie auf „Hinzufügen“ klicken, wird die Textspur beim nächsten Upload/Update zu den Videodaten hinzugefügt.
      1. Sprache - Der zweibuchstabige Code (gültiges Sprach-Tag BCP 47) für die Sprache der Textspur, zum Beispiel „en“ für Englisch.
      2. Label - Das Label für die Spur, die dem Benutzer angezeigt wird, z. B. in einem Menü, in dem die verschiedenen für Untertitel verfügbaren Sprachen aufgeführt sind.
      3. Art - Einer der fünf aufgelisteten unterstützten Track-Typen.
    4. Liste der hinzugefügten/in der Warteschlange stehenden Textspuren - Zeigt die Liste der vorhandenen Textspuren und neuen Textspuren an, die beim nächsten Speichern in die Warteschlange gestellt werden. Eine Textspur kann gelöscht werden, indem Sie auf den Link „Entfernen“ klicken und das Video speichern.

    Hinzufügen, Importieren und Löschen von Playlists und Experiences

    Der Abschnitt Playlists verwalten des Connectors wird verwendet, um neue Wiedergabelisten zu erstellen sowie vorhandene Wiedergabelisten zu durchsuchen und zu verwalten, einschließlich Bearbeitung und Aktualisierung. Im Gegensatz zu Videos werden Playlist-Informationen nicht in SharePoint gespeichert, sondern jedes Mal, wenn ein Konto in diesem Abschnitt durchsucht wird, aus Video Cloud abgerufen. Aus diesem Grund gibt es einige Verzögerungen, die mit der Verfügbarkeit von Playlist-Daten über die API verbunden sind, die in den folgenden Abschnitten aufgerufen wird.

    Landingpage „Playlists verwalten“

    Auf der Landing-Page Playlists verwalten wird die Liste der Felder angezeigt, mit denen Sie in den verfügbaren Konten nach Wiedergabelisten suchen oder neue erstellen können:

    • Konto auswählen - Diese Dropdown-Liste wird mit allen Konten gefüllt, für die der Benutzer autorisiert ist. Dies wird über die Zuweisung von Kontogruppen aktiviert. Der Benutzer muss ein Konto auswählen, um alle Playlist-Angebote anzuzeigen.
    • Verfügbare Wiedergabeliste - Um Wiedergabelisten zu durchsuchen, geben Sie Begriffe in dieses Feld ein.
    • Playlist-Anzeigesäulen - Playlist-Daten werden in diesen Spalten angezeigt. Die Ergebnisse können nach jeder Spalte sortiert werden, indem Sie auf die Spaltenüberschrift klicken.
    • Playlist-Auflistung - Jede Playlist-Liste enthält Folgendes:
      • Name der Wiedergabeliste - Wenn Sie auf den Namen klicken, wird die Wiedergabeliste zur Bearbeitung der Daten geöffnet.
      • Eine Liste von Video-IDs, die in der Wiedergabeliste aktiv sind.
      • Kurzbeschreibung.
      • Art der Wiedergabeliste - Manuelle oder Smart Playlist.
      • Die ID der Playlist (Video Cloud-Wiedergabelisten-ID).
      • Der Link „Playlist bearbeiten“. Dieser Link (oder das Klicken auf den Namen) wird verwendet, um die Wiedergabeliste zum Bearbeiten zu öffnen oder um ihre Details anzuzeigen.
    • Neue Playlist hinzufügen - Klicken Sie auf diese Schaltfläche, um eine neue Wiedergabeliste im ausgewählten Konto zu erstellen.

    Hinzufügen einer neuen manuellen Playlist

    Um eine neue Playlist zu einem Konto hinzuzufügen, klickt der Benutzer auf die Schaltfläche „Neue Playlist hinzufügen“. Dadurch wird das Formular Playlist hinzufügen geöffnet:

    Beachten Sie, dass Videos in der Wiedergabeliste nach oben und unten verschoben werden können, indem Sie das zu verschiedende Video auswählen und die Option „Video verschieben“ aktivieren, um die Schaltflächen nach oben/unten zu verwenden. Es können mehrere Videos gleichzeitig verschoben werden.

    • Name - Der Name der neuen Playlist.
    • Beschreibung - Eine kurze Beschreibung für die Playlist.
    • Typ - Der Typ der Wiedergabeliste. In diesem Fall sollte es auf Manuell eingestellt sein.
    • Verfügbare Videos - Anzeige des Suchfelds und der Liste der im Konto verfügbaren Videos. Auf diese Weise kann der Benutzer die Videooptionen einschränken, die der Wiedergabeliste hinzugefügt werden sollen.
    • Action Arrows - Verschiebt ausgewählte Videos in die Wiedergabeliste und aus dieser heraus.
    • Videos in dieser Playlist - Die aktuellen Videos in der Playlist werden hier angezeigt.
    • Erstellen - Übernehmen Sie die Änderungen und starten Sie den Prozess der Erstellung der neuen Wiedergabeliste.

    Hinzufügen einer neuen Smart-Playlist

    • Name - Der Name der neuen Playlist.
    • Beschreibung - Eine kurze Beschreibung für die Playlist.
    • Typ - Der Typ der Wiedergabeliste. In diesem Fall sollte es auf Smart gesetzt sein.
    • Tags - Für eine intelligente Wiedergabeliste kann der Benutzer ein oder mehrere Tags auswählen, die zum Filtern einer Wiedergabeliste verwendet werden können. Das erste Dropdown-Menü ermöglicht es dem Benutzer, „Eins oder mehr enthalten“ oder „Alle enthalten“ auszuwählen, und das zweite Dropdown-Menü ermöglicht es dem Benutzer, die Tags für die Smart Playlist einzugeben. Abhängig von ihrer Auswahl erscheinen entweder alle Videos mit einem der Tags oder nur diejenigen, die alle Tags enthalten.
    • Reihenfolge - Auf diese Weise kann der Benutzer festlegen, wie die Reihenfolge der passenden Videos festgelegt wird. Die Auswahl umfasst Folgendes:
      • Alphabetisch
      • Aktiviertes Datum (neuestes zuerst)
      • Aktiviertes Datum (ältestes zuerst)
      • Gesamtzahl der
      • Vergangene Woche spielt
      • Startdatum (ältestes zuerst)
      • Startdatum (neuestes zuerst)
    • Begrenzen auf - Die maximale Anzahl von Videos, die in die Wiedergabeliste aufgenommen werden sollen.
    • Vorschau - Klicken Sie hier, um die Wiedergabeliste so anzuzeigen, wie sie derzeit mit den ausgewählten Optionen funktioniert.
    • Videos in dieser Playlist - Die aktuellen Videos in der Playlist werden hier angezeigt.
    • Erstellen - Übernehmen Sie die Änderungen und starten Sie den Prozess der Erstellung der neuen Wiedergabeliste.

    Bearbeiten einer bestehenden Playlist

    Um eine vorhandene Wiedergabeliste zu bearbeiten, klicke entweder auf den Namen des Videos oder den Bearbeitungslink in der ganz rechten Spalte auf der Seite Playlists verwalten. Dadurch gelangen Sie zu einer Seite, die fast identisch mit der Seite „Neue Playlist hinzufügen“ ist. Anstelle einer Schaltfläche „Erstellen“ gibt es jedoch eine Schaltfläche „Speichern“. Fast alles kann bearbeitet werden, außer dem Playlist-Typ. Während Sie den Typ ändern und speichern können, werden die Änderungen nicht wirklich gespeichert, wenn Sie die Wiedergabeliste erneut anzeigen.

    Erlebnisse

    Um Erlebnisse anzuzeigen, klicken Sie auf den Tab Erfahrungen verwalten. Die verfügbaren Erlebnisse werden aufgelistet und verfügbar sein. Sie können sie anzeigen, indem Sie auf den Namen klicken, und die Erfahrung kann in Seiten eingebettet werden, sie kann jedoch nur von Brightcove Studio aus bearbeitet werden. Es besteht die Möglichkeit zu sehen, wie das Erlebnis sowohl auf einem Desktop als auch auf Mobilgeräten aussehen wird und wie es in verschiedenen Wiedergabestatus aussieht.

    Video zu einer SharePoint-Website hinzufügen

    Es gibt verschiedene Möglichkeiten, ein Video in eine SharePoint-Website einzubetten, je nachdem, ob Sie die moderne Benutzeroberfläche oder die klassische Benutzeroberfläche verwenden. Während sich der Einbettungsprozess unterscheidet, ist bei beiden UIs der Prozess zum Einbetten von Videos, Wiedergabelisten und Erlebnissen gleich. Klicke einfach auf das Brightcove Playlist oder Brightcove Experience Symbol anstelle des Brightcove Video-Symbols.

    Modernes UI

    Klicken Sie in der modernen Benutzeroberfläche zuerst auf das Pluszeichen, um ein neues App-Part an der gewünschten Stelle hinzuzufügen. Wählen Sie Brightcove Video aus.

    Dies öffnet ein Auswahlfenster. Wählen Sie ein Video aus und scrollen Sie dann nach unten und klicken Sie auf „

    Das Video wird nicht sofort angezeigt, aber wenn Sie auf „Veröffentlichen“ oder „Als Entwurf speichern“ klicken, wird die fertige Seite mit dem eingebetteten Video angezeigt.

    Klassisches UI

    Während die moderne Benutzeroberfläche nur die eine Möglichkeit zum Einbetten hat, bietet die klassische Benutzeroberfläche drei Optionen. Alle von ihnen stammen aus dem Menü „Einfügen“ ganz rechts in der oberen Symbolleiste.

    Einbetten

    Klicken Sie zuerst auf das Brightcove Video-Symbol. Oder klicken Sie auf diese, wenn Sie einen Spielisten oder ein Erlebnis einbetten. Der Prozess ist derselbe.

    Dadurch wird das Auswahlmenü für Brightcove Video geöffnet.

    Wähle das Video aus, scrolle dann nach unten und klicke auf „In die Zwischenablage kopieren“.

    Schließen Sie das Brightcove Video-Fenster und klicken Sie dann im Menü Einfügen auf „Einbetten“.

    Fügen Sie den kopierten Code in das Feld ein und klicken Sie dann auf Einfügen.

    Ihr Video wird auf der Seite erscheinen.

    App-Teil

    Um ein App-Teil hinzuzufügen, klicken Sie auf das App-Part Symbol.

    Dadurch wird ein Menü geöffnet, das Optionen für verschiedene App-Parts bietet. Wählen Sie die Option „Brightcove Video“.

    Dadurch wird der Seite ein Webpart hinzugefügt, aber zunächst ist kein Video angehängt.

    Markieren Sie das Kästchen und klicken Sie auf den Pfeil, um „Webpart bearbeiten“ auszuwählen. Dadurch werden die Brightcove Videos geöffnet und Sie können ein Video auswählen. Klicken Sie unten auf Übernehmen. Das Video wird angezeigt, sobald Sie die Seite speichern.

    Web-Teil

    Webparts können auf zwei Arten hinzugefügt werden. Wenn Sie auf das Webparte-Symbol klicken, wird ein Menü geöffnet, das dem App-Pass-Menü ähnelt, und wenn Sie dieselben Schritte ausführen, werden dieselben Ergebnisse angezeigt.

    Sie können auch erneut auf die Schaltfläche „Brightcove Video“ klicken und ein Video auswählen. Wählen Sie diesmal jedoch „Dowload-Webpart“.

    Eine Datei wird auf Ihren Computer heruntergeladen. Klicken Sie auf das Webpart-Symbol. Wenn das Menü geöffnet wird, laden Sie die Datei in das Feld „Webpart hochladen“ hoch. Klicken Sie auf Hochladen

    Ihr Video wird auf Ihrer Seite angezeigt.
