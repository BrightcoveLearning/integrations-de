Installieren von Brightcove Video Connect für die Video-Cloud Sitecore CMS

Dieses Thema behandelt den Installationsprozess für das Brightcove Video Connect-Modul.

 

Installieren Sie Video Cloud Sitecore Media Framework 2.1

Brightcove Video Connect erfordert Video-Cloud Sitecore Media Framework muss zuerst installiert werden. Das Video Cloud Sitecore Media Framework wird mit dem Video Cloud Sitecore installiert Installationsassistent. Befolgen Sie die Anweisungen des Installationsassistenten.

Freigeben von Seitenereignissen aus dem Workflow

Die Video-Cloud Sitecore Media Framework enthält eine Reihe von Seitenereignissen, die aus dem Workflow freigegeben und veröffentlicht werden müssen, bevor sie verwendet werden können.

  1. Öffne das Video-Cloud Sitecore Arbeitsbox
  2. Wählen Sie den Analytics-Workflow aus
  3. Stellen Sie die folgenden Seitenereignisse bereit:
    • Wiedergabe geändert
    • Wiedergabe abgeschlossen
    • Wiedergabefehler
    • Wiedergabe gestartet
  4. Veröffentlichen Sie die Seitenereignisse

Installieren Sie Brightcove Video Connect für die Video-Cloud Sitecore Erlebnisplattform

Brightcove Video Connect für Video-Cloud Sitecore wird installiert mit dem Video-Cloud Sitecore Installationsassistent. Befolgen Sie die Anweisungen des Installationsassistenten. Die Video-Cloud Sitecore Plug-in kann über einen Link in der obigen Tabelle heruntergeladen werden. Diese Zip-Datei enthält sowohl das Quell- als auch das Plug-In-Paket.  Erst entpacken und dann Laden Sie das interne Zip-Paket auf Ihr Video-Cloud Sitecore Beispiel.

Reporting-Datenbankkonfiguration

Damit die Video-Cloud Sitecore Damit Media Framework-Berichte funktionieren, muss der Erfahrungsdatenbank (xDB)-Berichtsdatenbank eine neue Faktentabelle hinzugefügt werden. Ein Skript, mit dem diese neue Tabelle erstellt wird, ist auf SDN verfügbar. Sie müssen dieses Skript in Ihrer xDB-Berichtsdatenbank ausführen. Dieser Schritt sollte von Ihrem Datenbankadministrator ausgeführt werden.

Hinweis: Es ist möglich, mehrere zu installieren Video-Cloud Sitecore Media Framework-Konnektoren. Dieses Skript muss nur einmal pro Berichtsdatenbank ausgeführt werden, unabhängig davon, wie viele verschiedene Connectors Sie installiert haben.

Multi-Server-Konfiguration

Video-Cloud Sitecore Media Framework kann in einer Content-Authoring-Umgebung mit mehreren Servern installiert werden. Wenn dein Video-Cloud Sitecore Umgebung auf diese Weise konfiguriert ist, müssen Sie Folgendes konfigurieren:

Content Management Server

Wenn Sie den Scheduler zum Synchronisieren von Medien verwenden, muss der Scheduler nur auf einem CM-Server (Content Management) konfiguriert sein. Entfernen Sie den geplanten Job aus den Konfigurationsdateien auf den anderen CM-Servern. Die Video-Cloud Sitecore Die Ereigniswarteschlange kümmert sich um die Synchronisierung der anderen CM-Server.

Hinweis: Das Konfigurieren des Schedulers auf mehreren CM-Servern verursacht kein Problem, hat jedoch keinen Vorteil.

Content Delivery Server

Entfernen Sie auf jedem Content Delivery (CD) -Server den folgenden Index aus der angegebenen Konfigurationsdatei. Dieser Index wird nur in der CM-Umgebung benötigt.

Datei: Video Cloud Sitecore .MediaFramework.Services.Brightcove.config
Indexname: MediaFramework_Brightcove_Index
 

Bei der Konfiguration von CD-Servern ist es möglicherweise praktisch, Konfigurationsdateien von einem CM-Server zu kopieren. Wenn Sie dies tun und einen Scheduler zum Synchronisieren von Medien hinzugefügt haben, sollten Sie den geplanten Job aus den Konfigurationsdateien auf den CD-Servern entfernen. Die Mediensynchronisierung sollte nicht auf den CD-Servern ausgeführt werden. Es sollte auf einem CM-Server ausgeführt werden. Durch das Veröffentlichen werden die Medienelemente auf den CD-Servern verfügbar gemacht.

Verknüpfung Video-Cloud Konten

In diesem Thema wird beschrieben, wie Sie ein Konto in erstellen Video-Cloud Sitecore das entspricht einem Brightcove Video-Cloud Konto.

 

Erhalten Sie Kontoanmeldeinformationen von Brightcove

Damit Media Framework eine Verbindung zu Brightcove Video Cloud herstellen kann, müssen die Kontoanmeldeinformationen von Brightcove zu Video Cloud Sitecore hinzugefügt werden . In diesem Abschnitt wird beschrieben, wie Sie die Kontoanmeldeinformationen in Brightcove suchen. Im folgenden Abschnitt wird beschrieben, wie Sie diese Anmeldeinformationen zu Video Cloud Sitecore hinzufügen.

  1. Melden Sie sich bei Brightcove Video Cloud an. Notieren Sie den Publisher-ID-Wert. Sie müssen dies eingeben in Video-Cloud Sitecore im folgenden Abschnitt.
  2. Navigieren Sie zu Admin.
  3. Navigieren Sie zur API-Authentifizierung.
  4. Registrieren Sie eine neue Bewerbung.
  5. Benennen Sie die Anwendung, legen Sie den Kontozugriff fest und aktivieren Sie APIs.
  6. Notieren Sie die Client-ID und das Client-Geheimnis. Sie müssen sie eingeben in Video-Cloud Sitecore im folgenden Abschnitt.

Brightcove-Konto hinzufügen zu Video-Cloud Sitecore

In diesem Abschnitt wird beschrieben, wie Sie Video Cloud Sitecore ein Brightcove-Konto hinzufügen.

  1. Loggen Sie sich in die ein Video-Cloud Sitecore Schreibtisch.
  2. Öffnen Sie die Medienbibliothek.
  3. Navigieren Sie zu Medienbibliothek> Media Framework> Konten.
  4. Fügen Sie ein neues Brightcove-Kontoelement hinzu.
  5. Geben Sie die im vorherigen Abschnitt angegebenen Werte für Publisher-ID, Client-ID und Client Secret ein.
  6. Speichern Sie den Artikel.

Definieren Video-Cloud Spieler

Ein Player wird benötigt, um ein einzelnes Video oder eine Wiedergabeliste mit Videos anzuzeigen. Ein Player definiert, welche Funktionen dem Besucher zur Verfügung stehen, der ein Video ansehen möchte. Video-Cloud Sitecore Media Framework ermöglicht es Inhaltsautoren, neue Player hinzuzufügen, vorhandene Player zu bearbeiten und vorhandene Player zu löschen.

 

Spieler hinzufügen

In diesem Abschnitt wird beschrieben, wie Sie die Video-Cloud Sitecore Gegenstand, der einem Spieler entspricht.

  1. Navigieren Sie in Brightcove Video Cloud zum Abschnitt Player.
  2. Wählen Sie den Player aus, den Sie in Video Cloud Sitecore erstellen möchten. Notieren Sie den Player-ID-Wert. Sie müssen dies eingeben in Video-Cloud Sitecore im folgenden Abschnitt.
  3. Wählen Sie in Video Cloud Sitecore in der Medienbibliothek den Eintrag Brightcove Account aus.
  4. Wählen Sie den Eintrag Konten.
  5. Wählen Sie den Gegenstand Spieler.
  6. Fügen Sie ein neues Brightcove Video Player-Element hinzu. Sie werden aufgefordert, die Spieler-ID einzugeben. Geben Sie den Wert aus Brightcove Video Cloud ein.

Standardspieler einstellen

Damit Videos und Wiedergabelisten in Video Cloud Sitecore in der Vorschau angezeigt werden können, müssen die Standardplayer angegeben werden. In diesem Abschnitt wird beschrieben, wie Sie die Standardspieler festlegen.

  1. Wählen Sie in Video Cloud Sitecore in der Medienbibliothek den Eintrag Brightcove Account aus.
  2. Wählen Sie den Eintrag Konten.
  3. Wählen Sie den Eintrag Einstellungen.
  4. Legen Sie die Werte für das Feld Standard-Video-Player und das Feld Standard-Playlist-Player fest.
  5. Speichern Sie den Artikel

Spieler bearbeiten

Spielerobjekte werden wie alle anderen bearbeitet Video-Cloud Sitecore Produkte.

Spieler löschen

Spielerobjekte werden wie alle anderen gelöscht Video-Cloud Sitecore Produkte.

Hinweis: Das Löschen eines Players aus Video Cloud Sitecore führt nicht dazu, dass der Player aus Brightcove Video Cloud gelöscht wird.

Videos importieren, hinzufügen, in der Vorschau anzeigen und löschen

Video-Cloud Sitecore Media Framework ermöglicht es einem Inhaltsautor, Videos von seinem lokalen Computer hochzuladen sowie vorhandene Videos zu bearbeiten und zu löschen. Neue Tags können definiert werden, um Videoinhalte zu organisieren.

 

Video importieren

EIN Video-Cloud Sitecore Inhaltsautor oder Administrator können den Medienimportprozess manuell auslösen.

Wenn Sie Videos importieren, spiegelt Video Cloud Sitecore den aktuellen Tag wider, an dem die Videos importiert wurden, anstatt das Upload-Datum innerhalb von Video-Cloud. Wenn Sie eine zuvor vorhandene Video Cloud Sitecore-Implementierung verschieben, können Sie den vorherigen Inhalt migrieren und der Konnektor erkennt diese Videos.

Das Brightcove Video Connect-Plug-In enthält einen Agenten, der Medien regelmäßig automatisch synchronisiert. Der Agent ist an folgendem Ort definiert:

Konfigurationsdatei: Video Cloud Sitecore .MediaFramework.Services.Brightcove.config
Agentenname: MediaFramework_Import_Brightcove
  1. Zeigen Sie das Media Framework-Menüband an, indem Sie mit der rechten Maustaste auf das Hauptmenü klicken und Media Framework auswählen.
  2. Klicken Sie auf das Media Framework-Menüband.
  3. Klicken Sie auf die Schaltfläche Alle importieren.
  4. Während des Importvorgangs wird ein Statusfeld angezeigt. Wenn das Statusfeld verschwindet, ist der Importvorgang abgeschlossen.
  5. Klicken Sie auf die Registerkarte Suchen, um zu bestätigen, dass Medien importiert wurden.
  6. Führen Sie eine Suche durch.
  7. Nach Videos filtern.
  8. Suchergebnisse werden angezeigt.

Video hinzufügen

Video-Cloud Sitecore Media Framework ermöglicht es Inhaltsautoren, neue Videos hinzuzufügen.

  1. Öffne das Video-Cloud Sitecore Medienbibliothek.
  2. Navigieren Sie zu dem Konto mit den Videos, für die Sie eine Wiedergabeliste erstellen möchten.
  3. Navigieren Sie zum Element Medieninhalt.
  4. Klicken Sie auf Brightcove Video.
  5. Klicken Sie auf Dateien hinzufügen.
  6. Wählen Sie die Datei aus, die Sie von Ihrem lokalen Computer hochladen möchten.
  7. Klicken Start.
  8. Schließen Sie nach dem Hochladen des Videos das Uploader-Fenster.

Tags definieren

Video-Cloud Sitecore Media Framework ermöglicht es Inhaltsautoren, neue Tags hinzuzufügen.

  1. Öffne das Video-Cloud Sitecore Medienbibliothek.
  2. Navigieren Sie zu dem Konto mit den Videos, für die Sie ein Tag erstellen möchten.
  3. Navigieren Sie zum Element Tags.
  4. Klicken Sie auf Brightcove-Tag.
  5. Geben Sie einen Artikelnamen ein. Dies wird zum Tag.

Video und Tags bearbeiten

Video- und Tag-Elemente werden wie alle anderen bearbeitet Video-Cloud Sitecore Produkte.

Video und Tags löschen

Video- und Tag-Elemente werden wie alle anderen gelöscht Video-Cloud Sitecore Produkte.

Suchen Sie nach Videos und Wiedergabelisten

Videos und Wiedergabelisten können gefunden werden, indem in ihrem Namen und ihren Beschreibungen nach Textzeichenfolgen gesucht wird. Videos können auch gefunden werden, indem in ihren benutzerdefinierten Feldern nach Textzeichenfolgen gesucht wird.

 

Suche

Klicken Sie auf die Registerkarte Suchen, um nach Medien zu suchen. Geben Sie den zu suchenden Text in die Namens- oder Beschreibungsfelder ein.

Klicken Sie auf die Lupe, um eine Suche durchzuführen.

Filtern

Suchergebnisse können nach Typ wie Video oder Wiedergabeliste, Datum und anderen Attributen gefiltert werden.

Suchergebnisse werden angezeigt.

Importieren, Hinzufügen, Vorschau und Löschen von Wiedergabelisten

Eine Wiedergabeliste ist eine Sammlung von Videos in einem einzelnen Player. Video-Cloud Sitecore Media Framework ermöglicht es Inhaltsautoren, neue Wiedergabelisten hinzuzufügen, vorhandene Wiedergabelisten zu bearbeiten und vorhandene Wiedergabelisten zu löschen.

Wenn Sie Wiedergabelisten importieren, spiegelt Video Cloud Sitecore den aktuellen Tag wider, an dem die Wiedergabelisten importiert wurden, anstatt das Erstellungsdatum innerhalb von Video-Cloud. Wenn Sie eine zuvor vorhandene Video Cloud Sitecore-Implementierung verschieben, können Sie den vorherigen Inhalt migrieren und der Konnektor erkennt diese Videos.
 

Playlist hinzufügen

Video-Cloud Sitecore Media Framework ermöglicht es Inhaltsautoren, neue Wiedergabelisten hinzuzufügen.

  1. Öffne das Video-Cloud Sitecore Medienbibliothek.
  2. Navigieren Sie zu dem Konto mit den Videos, für die Sie Wiedergabelisten erstellen möchten.
  3. Navigieren Sie zum Element Medieninhalt.
  4. Fügen Sie ein neues Brightcove-Wiedergabelistenelement hinzu.
  5. Geben Sie für das Feld Name einen beschreibenden Namen für das Video ein.
  6. Wählen Sie im Feld Video-IDs die Videos aus, die Sie in die Wiedergabeliste aufnehmen möchten.

Wiedergabeliste bearbeiten

Playlist-Elemente werden wie alle anderen bearbeitet Video-Cloud Sitecore Produkte.

Wiedergabeliste löschen

Playlist-Elemente werden wie alle anderen gelöscht Video-Cloud Sitecore Produkte.

Video hinzufügen zu Video-Cloud Sitecore Webseite

Video-Cloud Sitecore Media Framework ermöglicht es Inhaltsautoren, Medien auf verschiedene Weise in ihre Seiten einzufügen. Dieses Kapitel behandelt die verfügbaren Optionen.

 

Rich-Text-Editor

Mit Rich-Text-Feldern können Inhaltsautoren formatierten Inhalt in ihre Seiten einfügen. Inhaltsautoren verwenden den Rich-Text-Editor, um diesen formatierten Inhalt zu erstellen und zu bearbeiten. In diesem Abschnitt werden die Funktionen behandelt, die Video-Cloud Sitecore Media Framework ergänzt den Rich-Text-Editor.

Medien einbetten

Medien können direkt in ein Rich-Text-Feld eingebettet werden.

  1. Klicken Sie im Rich-Text-Editor auf die Schaltfläche Medien einbetten.
  2. Verwenden Sie den Suchbildschirm, um das Medium zu finden, das Sie einlegen möchten. Klicken Sie auf die Zeile, die das Medium darstellt, das Sie einfügen möchten. Klicken Sie auf die Schaltfläche Weiter>.
  3. Abhängig vom ausgewählten Medientyp werden zusätzliche Optionen angezeigt. Wenn Sie beispielsweise ein Video auswählen, müssen Sie den Player angeben, der zum Anzeigen des Videos verwendet werden soll, sowie die Abmessungen für den Player. Klicken Fertig.
  4. Das Medium ist in das Rich-Text-Feld eingebettet.

Medienlink einbetten

Ein Link zu Medien kann in ein Rich-Text-Feld eingefügt werden.

  1. Klicken Sie im Rich-Text-Editor auf die Schaltfläche Medienlink einbetten.
  2. Verwenden Sie den Suchbildschirm, um das Medium zu finden, das Sie einlegen möchten. Klicken Sie auf die Zeile, die das Medium darstellt, das Sie einfügen möchten. Klicken Sie auf die Schaltfläche Weiter>.
  3. Geben Sie einen Wert für den Link-Titel ein. Dies ist der Text, der für den Medienlink angezeigt wird.
    Darüber hinaus werden Ihnen je nach ausgewähltem Medientyp zusätzliche Optionen angezeigt. Wenn Sie beispielsweise ein Video auswählen, müssen Sie den Player angeben, der zum Anzeigen des Videos verwendet werden soll, sowie die Abmessungen für den Player. Klicken Fertig.
  4. Ein Link ist in das Rich-Text-Feld eingebettet.

Rendern

Mit Renderings können Inhalte an einer bestimmten Stelle auf einer Seite positioniert werden. Video-Cloud Sitecore Media Framework enthält ein Rendering, mit dem Inhaltsautoren Medien in Seiten einfügen können.

Medien-Rendering einbetten

  1. Verwenden Sie den Seiteneditor, um einem Platzhalter ein Rendering hinzuzufügen.
  2. Wählen Sie das Rendering "Medien einbetten".
  3. Das Rendering wurde dem Platzhalter hinzugefügt. Klicken Sie nun auf die Schaltfläche Medienelement einbetten.
  4. Verwenden Sie den Suchbildschirm, um das Medium zu finden, das Sie einlegen möchten. Klicken Sie auf die Zeile, die das Medium darstellt, das Sie einfügen möchten. Klicken Sie auf die Schaltfläche Weiter>.
  5. Abhängig vom ausgewählten Medientyp werden zusätzliche Optionen angezeigt. Wenn Sie beispielsweise ein Video auswählen, müssen Sie den Player angeben, der zum Anzeigen des Videos verwendet werden soll, sowie die Abmessungen für den Player. Klicken Fertig.
  6. Das Medium wurde für das Rendern ausgewählt.

Synchronisation zwischen Video-Cloud Sitecore und Video-Cloud

Dieser Abschnitt behandelt die Synchronisierung von Videos und Playlists zwischen Video-Cloud Sitecore für folgende Szenarien:

  • Wenn Videos und Wiedergabelisten aus Brightcove gelöscht werden
  • Wenn Videos und Playlists hinzugefügt wurden Video-Cloud Sitecore
  • Wenn Videos und Playlists gelöscht werden aus Video-Cloud Sitecore

Importieren von Brightcove nach Video-Cloud Sitecore war vorher abgedeckt Hier.

Video exportieren

Innerhalb von Video Cloud Sitecore wird über Video Cloud Sitecore auf Brightcove Medien zugegriffen Elemente, die während des Medienimport-Prozesses erstellt werden. Seit sie sind Video-Cloud Sitecore Elemente, ist es für Benutzer möglich, diese Elemente zu ändern. Der Medienexportprozess ist verantwortlich für den Export von Änderungen aus Video-Cloud Sitecore nach Brightcove.

Automatischer Medienexport

Wenn ein Video Cloud Sitecore-Element, das Brightcove-Medien entspricht, in der Video Cloud erstellt, gespeichert oder gelöscht wird Sitecore-Client, diese Änderungen werden an Brightcove weitergeleitet.

Echtzeit-Exporte

Wenn ein neues Medienelement in Video Cloud Sitecore erstellt wird, wird standardmäßig ein Echtzeit-Export ausgeführt. Dies bedeutet, dass die Exportaufgabe sofort ausgeführt wird.

Automatischen Export verhindern

Sie möchten möglicherweise keine Änderungen in Video-Cloud Sitecore nach Brightcove exportiert werden. Vielleicht möchten Sie die Verbindung von Video-Cloud Sitecore in Brightcove schreibgeschützt sein, in das Brightcove-Änderungen importiert werden Video-Cloud Sitecore aber wo Video-Cloud Sitecore Änderungen werden nicht nach Brightcove exportiert.

Um zu verhindern, dass Änderungen in Video-Cloud Sitecore nicht nach Brightcove exportiert werden soll, löschen Sie den Wert der folgenden Einstellung:

Konfigurationsdatei: Video Cloud Sitecore .MediaFramework.config
Einstellungsname: Video Cloud Sitecore .MediaFramework.ExportDatabases
 

Hinweis: Diese Einstellung gilt für die Video-Cloud Sitecore Media Framework, nicht nur das Brightcove Video Connect-Plug-In. Das Ändern dieser Einstellung gilt für alle Connectors und alle Konten.

Datenbereinigung

Bereinigungsprozesse werden verwendet, um Video-Cloud Sitecore und Brightcove-Daten synchron.

Video-Cloud Sitecore Aufräumen

Video-Cloud Sitecore Bereinigung ist der Prozess, der ausgeführt wird, wenn Medien in Brightcove und den entsprechenden Video-Cloud Sitecore Elemente müssen gelöscht werden. Da Brightcove nichts über Video Cloud Sitecore weiß, kann Brightcove Video Cloud Sitecore nicht benachrichtigen dass Medien gelöscht wurden. Deswegen, Video-Cloud Sitecore muss proaktiv sein.

Manuelle Bereinigung

EIN Video-Cloud Sitecore Inhaltsautor oder -administrator kann den Bereinigungsprozess manuell auslösen, indem er im Inhaltseditor auf die folgende Schaltfläche klickt:

Automatische Bereinigung

Das Brightcove Video Connect-Plug-In enthält einen Agenten, der den Bereinigungsprozess regelmäßig automatisch ausführt. Der Agent ist an folgendem Ort definiert:

Konfigurationsdatei: Video Cloud Sitecore .MediaFramework.Services.Brightcove.config
Agentenname: MediaFramework_Cleanup_Brightcove

Brightcove-Bereinigung

Die Brightcove-Bereinigung ist der Prozess, der ausgeführt wird, wenn ein Medienelement in gelöscht wird Video-Cloud Sitecore und die entsprechenden Medien in Brightcove müssen gelöscht oder deaktiviert werden. Da Brightcove weiß, dass das Medienelement gelöscht wurde, Video-Cloud Sitecore kann Brightcove benachrichtigen, dass die entsprechenden Medien in Brightcove gelöscht oder deaktiviert werden sollen.

Wenn ein Medienelement im Video Cloud Sitecore-Client gelöscht wird, wird ein Eintrag in die Exportwarteschlange vorgenommen. Der Vorgang ist der gleiche wie beim Speichern eines Medienelements.

Playback Event Analytics

Die Video-Cloud Sitecore Media Framework ermöglicht die Aufzeichnung von Videowiedergabeereignissen zur Verwendung in der Video-Cloud Sitecore Digitales Marketingsystem (DMS). Dieser Abschnitt behandelt die verfügbaren Optionen.

Wiedergabeereignisse

Wiedergabeereignisse werden verwendet, um DMS-Seitenereignisse als Ergebnis von Video-Player-Aktionen aufzuzeichnen. Wenn beispielsweise die Videowiedergabe startet, kann ein DMS-Seitenereignis geschrieben werden.

Standardwiedergabeereignisse

Video-Cloud Sitecore Media Framework enthält die folgenden Wiedergabeereignisse:

  • Wiedergabe gestartet
  • Wiedergabe abgeschlossen
  • Wiedergabefehler

Konfigurieren Sie Wiedergabeereignisse für das Konto

Standardmäßig werden die Ereignisse für die Wiedergabe gestartet und die Wiedergabe abgeschlossen. Dies ist im Element Einstellungen unter dem Element Brightcove-Konto aktiviert.

Das Element "Einstellungen" enthält ein Feld mit dem Namen "Regeln für Wiedergabeereignisse". Es ist eine Regel definiert, die das Media Framework anweist, die unter Videos und Wiedergabelisten angegebenen Ereignisse aufzuzeichnen.

Deaktivieren Sie Wiedergabeereignisse für Konto

Alle Wiedergabeereignisse können für ein bestimmtes Konto deaktiviert werden.

  1. Navigieren Sie im Inhaltseditor zu dem Konto unter Medienbibliothek> Media Framework> Konten.
  2. Wählen Sie den Eintrag Einstellungen.
  3. Das Feld Regeln für Wiedergabeereignisse beschreibt die Bedingungen, die erfüllt sein müssen, damit Wiedergabeereignisse registriert werden können. Standardmäßig lautet die Bedingung "wo wahr" und bedeutet "immer". Klicken Sie auf die Schaltfläche Regel bearbeiten, um die Bedingung zu ändern oder die vorhandene Regel zu entfernen.

Geben Sie Wiedergabeereignisse für Medien an

Jede Wiedergabeliste und jedes Videoelement verfügt über ein Feld mit dem Namen "Ereignisse". Dieses Feld bestimmt, welche Wiedergabeereignisse für das angegebene Element behandelt werden.

Erstellen Sie neue Wiedergabeereignisse

Sie können neue Wiedergabeereignisse hinzufügen, um detailliertere Informationen zur Videowiedergabe zu erfassen.

  1. Navigieren Sie im Inhaltseditor zu System> Module> Media Framework> Wiedergabeereignisse
  2. Fügen Sie mithilfe der Vorlage Wiedergabeereignis ein neues Element hinzu
  3. Wählen Sie im Feld Seitenereignis die Option Wiedergabe geändert
  4. Für das Feld Parameter können Sie einen Wert in einem der folgenden Formate eingeben:
    1. Nummer - Nachdem das Video in dieser Anzahl von Sekunden abgespielt wurde, wird das Seitenereignis aufgezeichnet. Ein Beispiel für einen gültigen Wert ist 15
    2. Prozent - Nachdem das Video in dieser Menge abgespielt wurde, wird das Seitenereignis aufgezeichnet. Ein Beispiel für einen gültigen Wert ist 50%
  5. Speichern Sie den Artikel
  6. Veröffentlichen Sie den Artikel
  7. Jetzt können Sie das neue Seitenereignis zu Ihrer Wiedergabeliste und Ihren Videos hinzufügen

Hinweis: Denken Sie daran, das neue Wiedergabeereignis Ihrer Wiedergabeliste und Ihren Videos zuzuweisen. Andernfalls wird das Ereignis nicht aufgezeichnet.

Die Video-Cloud Sitecore Media Framework enthält eine Reihe von Berichten, die sich auf die Videowiedergabe beziehen.

Wiedergabe gestartet

Dieser Bericht zeigt eine Liste der Videos, die das Ereignis "Video gestartet" ausgelöst haben.

Wiedergabe abgeschlossen

Dieser Bericht enthält eine Liste der Videos, die das Ereignis "Video abgeschlossen" ausgelöst haben.

Unvollständige Wiedergabe

Dieser Bericht enthält eine Liste der Videos, die das Ereignis "Video gestartet" ausgelöst haben, jedoch nicht das Ereignis "Video abgeschlossen".
