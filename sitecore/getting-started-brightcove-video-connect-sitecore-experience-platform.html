    Installieren von Brightcove Video Connect für das Video Cloud Sitecore CMS

    In diesem Thema wird der Installationsprozess für das Brightcove Video Connect-Modul behandelt.

     

    Installieren Video Cloud Sitecore Media Framework 2.1

    Brightcove Video Connect erfordert Video Cloud Sitecore Media Framework muss zuerst installiert werden. Das Video Cloud Sitecore Media Framework wird mit dem installiert Video Cloud Sitecore Installationsassistent. Befolgen Sie die Anweisungen des Installationsassistenten.

    Freigabe von Seitenereignissen aus

    Das Video Cloud Sitecore Media Framework enthält eine Reihe von Seitenereignissen, die aus dem Workflow freigegeben und veröffentlicht werden müssen, bevor sie verwendet werden können.

    1. Öffne das Video Cloud Sitecore Arbeitsbox
    2. Wählen Sie den Analytics-Work
    3. Stellen Sie die folgenden Seitenereignisse bereit:
      • Die Wiedergabe wurde geändert
      • Die Wiedergabe ist abgeschlossen
      • Wiedergabe-Fehler
      • Die Wiedergabe wurde gestartet
    4. Veröffentlichen Sie die Seitenereign

    Installieren Sie Brightcove Video Connect für das Video Cloud Sitecore Plattform erleben

    Brightcove Video Connect für Video Cloud Sitecore wird mit dem installiert Video Cloud Sitecore Installationsassistent. Befolgen Sie die Anweisungen des Installationsassistenten. Das Video Cloud Sitecore Das Plug-In kann über einen Link in der obigen Tabelle heruntergeladen werden. Diese ZIP-Datei enthält sowohl das Quell- als auch das Plug-in-Paket.  Entpacken Sie es zuerst und dann Laden Sie das interne Zip-Paket auf Ihre Video Cloud Sitecore Beispiel.

    Reporting-Datenbank-

    Damit die Video Cloud Sitecore Wenn Media Framework-Berichte funktionieren, muss der xDB-Berichtsdatenbank (Experience Database) eine neue Faktentabelle hinzugefügt werden. Ein Skript, das diese neue Tabelle erstellt, ist auf SDN verfügbar. Sie müssen dieses Skript in Ihrer xDB-Berichtsdatenbank ausführen. Dieser Schritt sollte von Ihrem Datenbankadministrator durchgeführt werden.

    Hinweis: Es ist möglich, mehrere zu installieren Video Cloud Sitecore Media Framework-Konnektoren. Dieses Skript muss nur einmal pro Berichtsdatenbank ausgeführt werden, unabhängig davon, wie viele verschiedene Konnektoren Sie installiert haben.

    Multi-Server-Konfiguration

    Video Cloud Sitecore Media Framework kann in einer Umgebung für die Erstellung von Inhalten mit mehreren Servern installiert werden. Wenn dein Video Cloud Sitecore Die Umgebung wird auf diese Weise konfiguriert. Sie müssen Folgendes konfigurieren:

    Content Management-Server

    Wenn Sie den Scheduler zum Synchronisieren von Medien verwenden, muss nur ein Content Management (CM) Server den Scheduler konfigurieren. Entfernen Sie den geplanten Auftrag aus den Konfigurationsdateien auf den anderen CM-Servern. Das Video Cloud Sitecore Die Ereigniswarteschlange übernimmt die Synchronisierung der anderen CM-Server.

    Hinweis: Die Konfiguration des Schedulers auf mehreren CM-Servern führt zu keinem Problem, aber das hat keinen Vorteil.

    Content Delivery Server

    Entfernen Sie auf jedem Content Delivery (CD) Server den folgenden Index aus der angegebenen Konfigurationsdatei. Dieser Index wird nur in der CM-Umgebung benötigt.

    Datei: Video Cloud Sitecore .MediaFramework.Services.Brightcove.config
    Bezeichnung des Index: MediaFramework_Brightcove_Index
     

    Bei der Konfiguration von CD-Servern ist es möglicherweise praktisch, Konfigurationsdateien von einem CM-Server zu kopieren. Wenn Sie dies tun und einen Scheduler zum Synchronisieren von Medien hinzugefügt haben, sollten Sie den geplanten Auftrag aus den Konfigurationsdateien auf den CD-Servern entfernen. Die Mediensynchronisierung sollte nicht auf den CD-Servern laufen. Es sollte auf einem CM-Server laufen. Publishing wird verwendet, um die Medienelemente auf den CD-Servern verfügbar zu machen.

    Video Cloud Konten verknüpfen

    In diesem Thema wird beschrieben, wie Sie ein Konto in erstellen Video Cloud Sitecore das entspricht einer Brightcove Video Cloud Konto.

     

    Holen Sie sich Konto-Zugangsdaten von Brightcove

    Damit Media Framework eine Verbindung zu Brightcove herstellen kann Video Cloud müssen die Kontoanmeldeinformationen von Brightcove hinzugefügt werden Video Cloud Sitecore . In diesem Abschnitt wird beschrieben, wie Sie die Kontoanmeldeinformationen in Brightcove finden. Im folgenden Abschnitt wird beschrieben, wie Sie diese Anmeldeinformationen hinzufügen Video Cloud Sitecore .

    1. Melden Sie sich bei Brightcove an Video Cloud . Notieren Sie den Wert der Herausgeber-ID Sie müssen dies eingeben Video Cloud Sitecore im folgenden Abschnitt.
    2. Navigieren Sie zu Admin.
    3. Navigieren Sie zu API-Authentifizierung.
    4. Registrieren Sie eine neue Anwendung.
    5. Benennen Sie die Anwendung, legen Sie den Kontozugriff fest und aktivieren Sie APIs.
    6. Notieren Sie sich die Client-ID und das Client-Geheimnis. Sie müssen sie eingeben Video Cloud Sitecore im folgenden Abschnitt.

    Brightcove-Konto hinzufügen zu Video Cloud Sitecore

    In diesem Abschnitt wird beschrieben, wie Sie ein Brightcove-Konto hinzufügen Video Cloud Sitecore .

    1. Melden Sie sich bei der Video Cloud Sitecore Desktop.
    2. Öffnen Sie die Medienbibliothek.
    3. Navigieren Sie zu Media Library > Media > Framework-
    4. Fügen Sie ein neues Brightcove Account-Element hinzu.
    5. Geben Sie die Werte für Herausgeber-ID, Client-ID und Client Secret ein, die im vorherigen Abschnitt angegeben sind.
    6. Speichern Sie den Artikel.

    Definieren von Video Cloud Spiel

    Ein Player wird benötigt, um ein einzelnes Video oder eine Wiedergabeliste mit Videos anzuzeigen. Ein Player definiert, welche Funktionen dem Besucher zur Verfügung stehen, der ein Video ansehen möchte. Video Cloud Sitecore Mit Media Framework können Inhaltsautoren neue Player hinzufügen, vorhandene Player bearbeiten und vorhandene Player löschen.

     

    Spieler hinzufügen

    In diesem Abschnitt wird beschrieben, wie Sie das manuell erstellen Video Cloud Sitecore Gegenstand, der einem Spieler entspricht.

    1. Navigieren Sie in Brightcove Video Cloud zum Abschnitt Spieler.
    2. Wählen Sie den Player aus, in dem Sie erstellen möchten Video Cloud Sitecore . Notieren Sie sich den Wert der Spieler-ID Sie müssen dies eingeben Video Cloud Sitecore im folgenden Abschnitt.
    3. Im Video Cloud Sitecore Wählen Sie in der Medienbibliothek das Element Brightcove-Konto aus.
    4. Wählen Sie den Punkt Konten aus.
    5. Wählen Sie den Gegenstand Spieler aus.
    6. Fügen Sie ein neues Brightcove Video Player-Objekt hinzu. Sie werden aufgefordert, die Spieler-ID einzugeben. Geben Sie den Wert von Brightcove ein Video Cloud .

    Standardspieler festlegen

    Damit Videos und Wiedergabelisten in der Vorschau angezeigt werden Video Cloud Sitecore müssen die Standardspieler angegeben werden. In diesem Abschnitt wird beschrieben, wie die Standardspieler festgelegt werden.

    1. Im Video Cloud Sitecore Wählen Sie in der Medienbibliothek das Element Brightcove-Konto aus.
    2. Wählen Sie den Punkt Konten aus.
    3. Wählen Sie den Punkt Einstellungen aus.
    4. Legen Sie die Werte für das Feld Default Video Player und das Feld Default Playlist Player fest.
    5. Speichern Sie den Artikel

    Spieler bearbeiten

    Spielerelemente werden wie alle anderen bearbeitet Video Cloud Sitecore Artikel.

    Spieler löschen

    Spielergegenstände werden wie alle anderen gelöscht Video Cloud Sitecore Artikel.

    Hinweis: Löschen eines Players aus Video Cloud Sitecore führt nicht dazu, dass der Player aus Brightcove gelöscht wird Video Cloud .

    Importieren, Hinzufügen, Vorschau und Löschen von Videos

    Video Cloud Sitecore Mit Media Framework kann ein Inhaltsautor Videos von seinem lokalen Computer hochladen sowie vorhandene Videos bearbeiten und löschen. Neue Tags können definiert werden, um Videoinhalte zu organisieren.

     

    Video importieren

    EIN Video Cloud Sitecore Der Autor oder Administrator von Inhalten kann den Medienimportprozess manuell auslösen.

    Wenn Sie Videos importieren, Video Cloud Sitecore zeigt den aktuellen Tag an, an dem die Videos importiert wurden, und nicht das Upload-Datum innerhalb von Video Cloud . Wenn Sie eine zuvor vorhandene verschieben Video Cloud Sitecore Implementierung können Sie den vorherigen Inhalt migrieren und der Connector erkennt diese Videos.

    Das Brightcove Video Connect Plug-In enthält einen Agenten, der Medien regelmäßig automatisch synchronisiert. Der Agent ist an folgendem Ort definiert:

    Config-Datei: Video Cloud Sitecore .MediaFramework.Services.Brightcove.config
    Name des Agenten: MediaFramework_Import_Brightcove
    1. Zeigen Sie das Media-Framework-Menüband an, indem Sie mit der rechten Maustaste auf das obere Menü klicken und
    2. Klicken Sie auf das Menüband Media Framework
    3. Klicken Sie auf die Schaltfläche „Alle importieren“.
    4. Während der Importvorgang läuft, wird ein Statusfeld angezeigt. Wenn das Statusfeld verschwindet, ist der Importvorgang abgeschlossen.
    5. Um zu bestätigen, dass Medien importiert wurden, klicken Sie auf die Registerkarte Suchen.
    6. Führen Sie eine Suche durch.
    7. Filtere nach Videos.
    8. Die Suchergebnisse werden angezeigt.

    Video hinzufügen

    Video Cloud Sitecore Mit Media Framework können Inhaltsautoren neue Videos hinzufügen.

    1. Öffne das Video Cloud Sitecore Medienbibliothek.
    2. Navigiere zu dem Konto mit den Videos, für die du eine Wiedergabeliste erstellen möchtest.
    3. Navigieren Sie zum Element Medieninhalt.
    4. Klicken Sie auf Brightcove Video.
    5. Klicken Sie auf „Dateien hinzufügen“
    6. Wählen Sie die Datei aus, die Sie von Ihrem lokalen Computer hochladen möchten.
    7. Klicken Sie auf Start.
    8. Nachdem das Video hochgeladen wurde, schließen Sie das Uploader-Fenster.

    Kennzeichen definieren

    Video Cloud Sitecore Mit Media Framework können Inhaltsautoren neue Tags hinzufügen.

    1. Öffne das Video Cloud Sitecore Medienbibliothek.
    2. Navigieren Sie zu dem Konto mit den Videos, für die Sie ein Tag erstellen möchten.
    3. Navigieren Sie zum Element „Tags“.
    4. Klicken Sie auf Brightcove Tag.
    5. Geben Sie einen Artikelnamen ein. Dies wird zum Tag.

    Video und Schlagwörter editieren

    Video- und Tag-Elemente werden wie alle anderen bearbeitet Video Cloud Sitecore Artikel.

    Video und Schlagwörter löschen

    Video- und Tag-Elemente werden wie alle anderen gelöscht Video Cloud Sitecore Artikel.

    Suche

    Klicken Sie auf die Lupe, um eine Suche durchzuführen.

    Filtern

    Die Suchergebnisse können nach Typ wie Video oder Wiedergabeliste nach Datum und nach anderen Attributen gefiltert werden.

    Die Suchergebnisse werden angezeigt.

    Importieren, Hinzufügen, Vorschau und Löschen von Playlisten

    Eine Playlist ist eine Sammlung von Videos in einem einzelnen Spieler. Video Cloud Sitecore Mit Media Framework können Inhaltsautoren neue Wiedergabelisten hinzufügen, vorhandene Wiedergabelisten bearbeiten und vorhandene Wiedergabelisten löschen.

    Wenn Sie Wiedergabelisten importieren, Video Cloud Sitecore zeigt den aktuellen Tag an, an dem die Wiedergabelisten importiert wurden, und nicht das Erstellungsdatum innerhalb von Video Cloud . Wenn Sie eine zuvor vorhandene verschieben Video Cloud Sitecore Implementierung können Sie den vorherigen Inhalt migrieren und der Connector erkennt diese Videos.
     

    Playlist hinzufügen

    Video Cloud Sitecore Mit Media Framework können Inhaltsautoren neue Wiedergabelisten hinzufügen.

    1. Öffne das Video Cloud Sitecore Medienbibliothek.
    2. Navigiere zu dem Konto mit den Videos, für die du eine Wiedergabeliste erstellen möchtest.
    3. Navigieren Sie zum Element Medieninhalt.
    4. Fügen Sie einen neuen Gegenstand der Brightcove Playlist hinzu.
    5. Geben Sie für das Feld Name einen beschreibenden Namen für das Video ein.
    6. Wählen Sie im Feld „Video-IDs“ die Videos aus, die Sie in die Wiedergabeliste aufnehmen möchten.

    Playlist bearbeiten

    Wiedergabelistenelemente werden wie alle anderen bearbeitet Video Cloud Sitecore Artikel.

    Playlist löschen

    Video hinzufügen zu Video Cloud Sitecore Webseite

    Video Cloud Sitecore Mit Media Framework können Inhaltsautoren Medien auf verschiedene Weise in ihre Seiten einfügen. In diesem Kapitel werden die verfügbaren Optionen behandelt.

     

    Rich-Text-Editor

    Rich-Text-Felder ermöglichen es Inhaltsautoren, formatierte Inhalte in ihre Seiten einzufügen. Inhaltsautoren verwenden den Rich-Text-Editor, um diesen formatierten Inhalt zu erstellen und zu bearbeiten. Dieser Abschnitt behandelt die Funktionen, die Video Cloud Sitecore Media Framework erweitert den Rich-Text-Editor.

    Medien einbetten

    Medien können direkt in ein Rich-Text-Feld eingebettet werden.

    1. Klicken Sie im Rich-Text-Editor auf die Schaltfläche Medien einbetten.
    2. Verwenden Sie den Suchbildschirm, um die Medien zu finden, die Sie einfügen möchten. Klicken Sie auf die Zeile, die die einzufügenden Medien darstellt. Klicken Sie auf die > Schaltfläche Weiter.
    3. Abhängig von der Art der ausgewählten Medien werden zusätzliche Optionen angezeigt. Wenn Sie beispielsweise ein Video auswählen, müssen Sie den Player angeben, der zur Anzeige des Videos verwendet werden soll, zusammen mit den Abmessungen für den Player. Klicken Sie auf Finish (Fertigstellen).
    4. Die Medien sind in das Rich-Text-Feld eingebettet.

    Medien-Link einbetten

    Ein Link zu Medien kann in ein Rich-Text-Feld eingefügt werden.

    1. Klicken Sie im Rich-Text-Editor auf die Schaltfläche Medienlink einbetten.
    2. Verwenden Sie den Suchbildschirm, um die Medien zu finden, die Sie einfügen möchten. Klicken Sie auf die Zeile, die die einzufügenden Medien darstellt. Klicken Sie auf die > Schaltfläche Weiter.
    3. Geben Sie einen Wert für Link-Titel ein. Dies ist der Text, der für den Medienlink angezeigt wird.
      Darüber hinaus sehen Sie je nach ausgewähltem Medientyp zusätzliche Optionen. Wenn Sie beispielsweise ein Video auswählen, müssen Sie den Player angeben, der zur Anzeige des Videos verwendet werden soll, zusammen mit den Abmessungen für den Player. Klicken Sie auf Finish (Fertigstellen).
    4. Ein Link ist in das Rich-Text-Feld eingebettet.

    Rendern

    Renderings ermöglichen die Positionierung von Inhalten an einem bestimmten Ort auf einer Seite. Video Cloud Sitecore Media Framework enthält ein Rendering, mit dem Inhaltsautoren Medien in Seiten einfügen können.

    Medien-Rendering einbetten

    1. Verwenden Sie den Seiteneditor, um einem Platzhalter ein Rendering hinzuzufügen.
    2. Wählen Sie das Rendering für Medien einbetten aus.
    3. Das Rendering wurde dem Platzhalter hinzugefügt. Klicken Sie nun auf die Schaltfläche Medienelement einbetten.
    4. Verwenden Sie den Suchbildschirm, um die Medien zu finden, die Sie einfügen möchten. Klicken Sie auf die Zeile, die die einzufügenden Medien darstellt. Klicken Sie auf die > Schaltfläche Weiter.
    5. Abhängig von der Art der ausgewählten Medien werden zusätzliche Optionen angezeigt. Wenn Sie beispielsweise ein Video auswählen, müssen Sie den Player angeben, der zur Anzeige des Videos verwendet werden soll, zusammen mit den Abmessungen für den Player. Klicken Sie auf Finish (Fertigstellen).
    6. Die Medien wurden für das Rendering ausgewählt.

    Synchronisation zwischen Video Cloud Sitecore und Video Cloud

    In diesem Abschnitt wird beschrieben, wie Videos und Wiedergabelisten synchronisiert werden Video Cloud Sitecore für die folgenden Szenarien:

    • Wenn Videos und Playlists aus Brightcove gelöscht werden
    • Wenn Videos und Wiedergabelisten hinzugefügt werden Video Cloud Sitecore
    • Wenn Videos und Wiedergabelisten aus gelöscht werden Video Cloud Sitecore

    Importieren von Brightcove nach Video Cloud Sitecore wurde zuvor abgedeckt Hier.

    Exportieren von Video

    Innerhalb Video Cloud Sitecore Auf Brightcove-Medien wird mit zugegriffen Video Cloud Sitecore Elemente, die während des Medienimportprozesses erstellt werden. Seit sie sind Video Cloud Sitecore Elemente können Benutzer diese Elemente ändern. Der Medienexportprozess ist für den Export von Änderungen aus verantwortlich Video Cloud Sitecore nach Brightcove.

    Automatischer Medienexport

    Wenn ein Video Cloud Sitecore Ein Element, das Brightcove-Medien entspricht, wird im erstellt, gespeichert oder gelöscht Video Cloud Sitecore Client werden diese Änderungen an Brightcove übertragen.

    Echtzeit-Exporte

    Wenn ein neues Medienelement in erstellt wird Video Cloud Sitecore Standardmäßig wird ein Echtzeit-Export durchgeführt. Dies bedeutet, dass die Exportaufgabe sofort ausgeführt wird.

    Automatisches Exportieren verhindern

    Möglicherweise möchten Sie keine Änderungen in Video Cloud Sitecore nach Brightcove exportiert werden. Möglicherweise möchten Sie die Verbindung von Video Cloud Sitecore in Brightcove, um schreibgeschützt zu sein, in das Brightcove-Änderungen importiert werden Video Cloud Sitecore aber wo Video Cloud Sitecore Änderungen werden nicht nach Brightcove exportiert.

    Um Änderungen in zu verhindern Video Cloud Sitecore Löschen Sie beim Exportieren nach Brightcove den Wert der folgenden Einstellung:

    Config-Datei: Video Cloud Sitecore .MediaFramework.config
    Namen der Einstellung: Video Cloud Sitecore .MediaFramework.ExportDatabases
     

    Hinweis: Diese Einstellung gilt für die Video Cloud Sitecore Media Framework, nicht nur das Brightcove Video Connect-Plug-In. Die Änderung dieser Einstellung gilt für alle Konnektoren und alle Konten.

    Daten-Cleanup

    Bereinigungsprozesse werden verwendet, um zu halten Video Cloud Sitecore und Brightcove-Daten synchron.

    Video Cloud Sitecore Aufräumen

    Video Cloud Sitecore Bereinigung ist der Prozess, der ausgeführt wird, wenn Medien in Brightcove und den entsprechenden Medien gelöscht werden Video Cloud Sitecore Artikel müssen gelöscht werden. Da weiß Brightcove nichts davon Video Cloud Sitecore , Brightcove kann nicht benachrichtigen Video Cloud Sitecore Dieses Medium wurde gelöscht. Deshalb, Video Cloud Sitecore muss proaktiv sein.

    Manuelle Cleanup

    EIN Video Cloud Sitecore Der Autor oder Administrator von Inhalten kann den Bereinigungsprozess manuell auslösen, indem er im Inhaltseditor auf die folgende Schaltfläche klickt:

    Automatisches Bereinigen

    Das Brightcove Video Connect Plug-In enthält einen Agenten, der den Bereinigungsprozess regelmäßig automatisch ausführt. Der Agent ist an folgendem Ort definiert:

    Config-Datei: Video Cloud Sitecore .MediaFramework.Services.Brightcove.config
    Name des Agenten: MediaFramework_Cleanup_Brightcove

    Brightcove Bereinigung

    Die Brightcove-Bereinigung wird ausgeführt, wenn ein Medienelement in gelöscht wird Video Cloud Sitecore und die entsprechenden Medien in Brightcove müssen gelöscht oder deaktiviert werden. Da Brightcove weiß, dass das Medienelement gelöscht wurde, Video Cloud Sitecore kann Brightcove benachrichtigen, dass das entsprechende Medium in Brightcove gelöscht oder deaktiviert werden soll.

    Wenn ein Medienelement in der gelöscht wird Video Cloud Sitecore Client wird ein Eintrag in der Exportwarteschlange vorgenommen. Der Vorgang ist derselbe wie beim Speichern eines Medienelements.

    Wiedergabe von Ereignisanaly

    Das Video Cloud Sitecore Mit Media Framework können Videowiedergabeereignisse zur Verwendung in der aufgezeichnet werden Video Cloud Sitecore Digitales Marketing-System (DMS). In diesem Abschnitt werden die verfügbaren Optionen behandelt.

    Wiedergabe-Ereignisse

    Wiedergabeereignisse werden verwendet, um DMS-Seitenereignisse als Ergebnis von Videoplayer-Aktionen aufzuzeichnen. Wenn beispielsweise die Videowiedergabe beginnt, kann ein DMS-Seitenereignis geschrieben werden.

    Standard-Wiedergabe-Ereignisse

    Video Cloud Sitecore Media Framework enthält die folgenden Wiedergabeereignisse:

    • Die Wiedergabe wurde gestartet
    • Die Wiedergabe ist abgeschlossen
    • Wiedergabe-Fehler

    Konfigurieren von Wiedergabe-Events für Konto

    Standardmäßig werden die gestartete Wiedergabe und abgeschlossene Wiedergabe-Ereignisse aufgezeichnet. Dies ist für den Punkt Einstellungen unter dem Element Brightcove Account aktiviert.

    Das Element Einstellungen hat ein Feld namens Wiedergabe-Ereignisregeln. Es wird eine Regel definiert, die das Media Framework anweist, die unter Videos und Playlists angegebenen Ereignisse aufzuzeichnen.

    Deaktivieren Sie Playback-Ereignisse für

    Alle Wiedergabeereignisse können für ein bestimmtes Konto deaktiviert werden.

    1. Navigieren Sie im Content Editor zu dem Konto unter Media Library > Media > Framework-Konten.
    2. Wählen Sie den Punkt Einstellungen aus.
    3. Das Feld Playback Events Rules beschreibt die Bedingungen, die erfüllt sein müssen, damit Playback-Events registriert werden können. Standardmäßig ist die Bedingung „wo zutreffend“, was „immer“ bedeutet. Klicken Sie auf die Schaltfläche „Regel bearbeiten“, um die Bedingung zu ändern oder die vorhandene Regel zu entfernen.

    Geben Sie Wiedergabe-Ereignisse für Medien

    Jede Wiedergabeliste und jedes Videoelement hat ein Feld mit dem Namen Events. Dieses Feld legt fest, welche Wiedergabeereignisse für das angegebene Element behandelt werden.

    Der folgende Screenshot zeigt an, dass Wiedergabeereignisse für das Video angegeben wurden:

    Hinweis: Standardwerte können verwendet werden, um sicherzustellen, dass bestimmte Ereignisse für alle Wiedergabelisten und Videos aufgezeichnet werden, aber das Feld ermöglicht es auch, dass einzelne Wiedergabelisten und Videos ihre eigenen Ereignisse auswählen.

    Hinweis: Diese Ereignisse werden nur aufgezeichnet, wenn das Element „Einstellungen“ unter dem Brightcove-Kontoelement das Feld „Wiedergabeereignisregeln“ mit der Aktion „Media-Element-Ereignisse hinzufügen“ konfiguriert ist. Siehe Abschnitt 3.1.2.

    Neue Playback-Events erstellen

    Sie können neue Wiedergabeereignisse hinzufügen, um detailliertere Informationen zur Videowiedergabe zu erfassen.

    1. Navigieren Sie im Inhalts-Editor zu > Systemmodule > Media > Framework-Wieder
    2. Fügen Sie ein neues Element mit der Vorlage Playback-Ereignis hinzu
    3. Wählen Sie für das Feld Seitenereignis die Option Wiedergabe geändert
    4. Für das Feld Parameter können Sie einen Wert in einem der folgenden Formate eingeben:
      1. Nummer - nachdem das Video diese Anzahl von Sekunden abgespielt wurde, wird das Seitenereignis aufgezeichnet. Ein Beispiel für einen gültigen Wert ist 15
      2. Prozent - nachdem das Video diesen Betrag wiedergegeben hat, wird das Seitenereignis aufgezeichnet. Ein Beispiel für einen gültigen Wert ist 50%
    5. Speichern Sie den Artikel
    6. Veröffentlichen Sie den Artikel
    7. Jetzt können Sie das neue Seitenereignis zu Ihrer Playlist und Ihren Videos hinzufügen

    Hinweis: Denken Sie daran, das neue Wiedergabeereignis Ihrer Playlist und Ihren Videos zuzuweisen. Andernfalls wird das Ereignis nicht aufgezeichnet.

    Berichte

    Das Video Cloud Sitecore Media Framework enthält eine Reihe von Berichten zur Videowiedergabe.

    Die Wiedergabe wurde gestartet

    Dieser Bericht zeigt eine Liste der Videos, die das Ereignis zum Starten des Videos ausgelöst haben.

    Abgeschlossene Wiedergabe

    Dieser Bericht zeigt eine Liste der Videos, die das abgeschlossene Videoereignis ausgelöst haben.

    Unvollständige Wiedergabe

    Dieser Bericht zeigt eine Liste der Videos, die das Videostart-Ereignis ausgelöst haben, aber nicht das Ereignis mit dem Abschluss des Videos.
