Einführung

Rampe AltitudeCDN ™ OmniCache Das Produkt ist ein Unternehmensvideoverteilungssystem, das die Proxy-Cache-Technologie verwendet, um den Videoverkehr über das Internet-Gateway oder Verbindungen mit begrenzter Bandbreite innerhalb des Unternehmensnetzwerks zu reduzieren. Mit OmniCache ziehen Client-Desktop-Computer und mobile Geräte, die den Brightcove Player verwenden, den Stream direkt vom OmniCache-Server und anschließend vom OmniCache-Server der Stream vom Brightcove-CDN, falls noch keine lokale Kopie im Cache vorhanden ist. OmniCache ist eine vollständig lokale Lösung, die als Software-Appliance geliefert wird und von der AltitudeCDN ™ Altimeter-Verwaltungskonsole verwaltet wird.  Omnicache kann eigenständig oder zusammen mit verwendet werden AltitudeCDN ™ Multicast + für gemischte Unicast- / Multicast-Netzwerke.

Die Integration mit der Brightcove verwendet die OmniCache Video.js Plug-In um die Stream-URL auf den OmniCache-Server umzuschreiben.  Ausführliche Informationen zum Einrichten des OmniCache-Servers für die Verwendung mit Video Cloud finden Sie hier TechNote von der Rampe.

Bevor Sie mit den folgenden Anweisungen fortfahren, führen Sie bitte den OmniCache-Schnellstartprozess mit dem Ramp-Team durch, um sicherzustellen, dass der OmniCache vorhanden ist Server und das lokale Netzwerk sind ordnungsgemäß konfiguriert.

Player-Konfiguration

Konfigurieren Sie zunächst einen Brightcove-Player für die Verwendung des OmniCache-Plug-Ins. Erstellen Sie einen neuen Player mit dem gewünschten Skin und den gewünschten Optionen.

Fügen Sie dann das Plug-In OmniCache Video.js hinzu.

Von hier aus kopieren und einfügen:

    https://livetools.ramp.com/omnicache/plugin/brightcove-omnicache-v1.4.js

Legen Sie abschließend die OmniCache-Plug-In-Optionen fest. Ersetzen Sie den Proxy-Wert "omnicachedemo.ramp.com" durch den DNS-Namen Ihres OmniCache-Servers. Alternativ können Sie auch einen DNS-Lastenausgleich angeben, um den Videoverkehr auf mehrere OmniCache-Server zu verteilen.

Von hier aus kopieren und einfügen:

    {
        "proxy": "omnicachedemo.ramp.com",
        "verbose": true,
        "timeout": 2000
    }

Brightcove-Player einbetten

Betten Sie den Brightcove-Player schließlich mithilfe des Standard-Player-Veröffentlichungsprozesses in Ihre Webseite ein.
