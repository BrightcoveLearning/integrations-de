    Benutzerhandbuch für kollective eCDN

    In diesem Thema erfahren Sie, wie Sie Ihre VOD- und Live-Inhalte mit der Kollective Syndicator-Anwendung synchronisieren.

    Übersicht

    Brightcove hat eine Integration ins Leben gerufen, um Brightcove Players mithilfe des Kollective eCDNVideo On Demand (VOD) und Brightcove Live-Events zur Verfügung zu stellen.

    Diese Integration besteht aus zwei Teilen:

    • Die Backend-Anwendung, die verwendet wird, um Videos von Brightcove zu Kollective zu synchronisieren

    • Das Kollective-Plugin für Brightcove Player, das die Wiedergabe von Video-Assets mit dem Kollective eCDN ermöglicht

    Dieses Thema konzentriert sich auf die serverseitige App von Kollective, mit der Ihre Inhalte synchronisiert werden.

    Prozessablauf

    Um VOD- oder Live-Inhalte abzuspielen, verwenden Sie den folgenden Prozessablauf:

    1. Wählen Sie entweder das VOD- oder Live-Asset in der Syndicator-App aus.

    2. Synchronisieren Sie Ihre Content-Assets mit Kollective mithilfe der Syndicator-App.

    3. Betten Sie den Brightcove Player mit dem Kollective-Plugin in Ihre Webseite ein, um Ihre Inhalte wiederzugeben.

    Erste Schritte

    Um Inhalte zu synchronisieren, führen Sie die folgenden Schritte aus:

    1. Wenn Sie dies noch nicht getan haben, befolgen Sie die Anweisungen zur Einrichtung im Kollective eCDN Deployment Guide.

    2. Überprüfen von Benutzerrollen

    3. Informationen zum Synchronisieren Ihrer VOD- und Live-Inhalte finden Sie in den folgenden Themen:

      • Synchronisieren von Videos
      • Synchronisieren von Live-Events
    4. Syndikator-Protokolle prüfen
    5. Einbetten des Spielercodes

    Überprüfen von Benutzerrollen

    Die Backend-Syndikator-Applikation von Kollective unterstützt zwei verschiedene Benutzerrollen:

    • Admin-Rolle - Admin-Benutzer können andere Benutzer erstellen (Admin- und Benutzerrollen), auf Unternehmenseinstellungen zugreifen, das Syndikator-Tool ausführen und die Synchronisationsprotokolle überprüfen. Sie haben Zugriff auf alle Bereiche der Syndikator-App.

    • Benutzerrolle - Diese Benutzer können das Syndikator-Tool ausführen und die Synchronisationsprotokolle überprüfen, jedoch keine Unternehmenseinstellungen bearbeiten. Sie haben Zugang zu Video Syndicator, Live Events Syndicator und die Syndicator-Protokolle , aber sie haben keinen Zugang zu Benutzer verwalten oder die Einstellungen Seiten.

    Admin-Benutzer

    Der Admin-Benutzer kann andere Benutzer (Admin- und Benutzerrollen) erstellen, auf Unternehmenseinstellungen zugreifen, das Syndikator-Tool ausführen und die Synchronisationsprotokolle überprüfen.

    Kontoeinstellungen

    Der Admin-Benutzer hat Zugriff auf den Abschnitt Einstellungen , um Kontoinformationen festzulegen. Gehen Sie folgendermaßen vor, um Kontoeinstellungen zu verwalten:

    1. Melden Sie sich als Admin-Benutzer bei der Kollective Syndicator App an.

    2. Wählen Sie in der linken Navigation die Option Einstellungenaus.

    3. Im Abschnitt Einstellungen können Sie Informationen über das Konto Ihres Unternehmens eingeben.

      Hier sind die Felddetails:

      • Name des Mieters - Ihr Firmenname

      • Beschreibung - Ihre Beschreibung

      • Kontaktname - Ihr Kontaktname

      • Kontakt-E-Mail - Ihre Kontakt-E-Mail-Adresse

      • Brightcove Konto-ID - Sie können dies von Brightcoves Video Cloud Studio erhalten

      • Brightcove Client Key - Sie können dies von Brightcoves Video Cloud Studio erhalten

        Brightcove Client Secret - Sie können dies von Brightcoves Video Cloud Studio erhalten

        Damit Kollective auf Ihre Inhalte in Brightcove zugreifen kann, müssen Sie Ihre Kundenanmeldeinformationen angeben. Sie können sie im Admin-Bereich von Video Cloud Studio generieren. Sie müssen die folgenden Berechtigungen auswählen:

        • CMS: Playlist lesen, Playlist Lesen/Schreiben, Video lesen, Video lesen/schreiben
        • Dynamischer Aufnahme: Erstellen, Push-Dateien
        • Spieler: Lesen, Lesen/Schreiben
        • SSAI: Lesen, Lesen/Schreiben

        Weitere Informationen zum Abrufen Ihres Brightcove Client Key und des Kundengeheimnisses finden Sie unter den folgenden Themen:

        • Verwalten von Anmeldedaten für API-
        • Alternativ können Sie dieses Tool verwenden: Client-Anmeldedaten erstellen

      • Brightcove Live API Key - Wenden Sie sich an den Brightcove Support

      • Kollective Mandanten-ID - Sie erhalten dies von Ihrem Account Manager

      • Kollective Service Token - Sie erhalten dies von Ihrem Account Manager

      • Automatische Synchronisierung zulassen - Wenn Sie möchten, dass alle Ihre Inhalte zu einem bestimmten Zeitpunkt synchronisiert werden, überprüfen Sie dies. Ansonsten lassen Sie es unkontrolliert. Einzelheiten finden Sie im nächsten Schritt.

    4. Wenn Sie Automatische Synchronisierung zulassen auswählen, werden die folgenden Details angezeigt:

      Dadurch kann der Synchronisationsauftrag automatisch im konfigurierbaren Zeitintervall ausgeführt werden.

      • Geben Sie ein Zeitintervall ein, in dem der Auftrag ausgeführt werden soll. (Beispiel: Führen Sie den Job alle 1 Stunde aus).
      • Wählen Sie die Zeit aus, zu der der Auftrag zum ersten Mal ausgeführt werden soll.
      • Geben Sie bei jeder Ausführung des Auftrags die zu synchronisierenden Maximalvideos ein.
      • Sie können auf die Schaltfläche „ Job jetzt ausführen “ klicken, um den Job sofort auszuführen.

    5. Klicken Sie auf die Schaltfläche Aktualisieren , um die Einstellungen zu speichern.

    Verwalten von Benutzern

    Mit einem Admin-Konto können Sie Benutzer erstellen, bearbeiten und löschen.

    1. Melden Sie sich als Admin-Benutzer bei der Kollective Syndicator App an.

    2. Wählen Sie in der linken Navigation Benutzer verwaltenaus.

    3. Sie sehen eine Liste aller Benutzer im Unternehmenskonto.

    Hinzufügen von Benutzern

    Sie können einen Benutzer mit einer Admin- oder einer Benutzerrolle hinzufügen.

    1. Klicken Sie auf der Seite „Benutzer verwalten “ auf die Schaltfläche Benutzer hinzufügen .

    2. Geben Sie die Informationen ein, um einen Benutzer mit einer Admin- oder Benutzerrolle zu erstellen.

    Benutzer bearbeiten

    Sie können die Informationen für einen bestehenden Benutzer bearbeiten.

    1. Wählen Sie auf der Seite „Benutzer verwalten “ einen Benutzer aus der Liste aus.

    2. Sie können jedes der Benutzerfelder mit Ausnahme von E-Mailaktualisieren.

    Nutzer löschen

    Sie können Benutzer aus der Benutzerliste löschen.

    1. Wählen Sie auf der Seite „ Benutzer verwalten “ einen Benutzer aus der Liste aus (mit Ausnahme des Benutzers, mit dem Sie sich angemeldet haben).

    2. Wählen Sie die Schaltfläche „ Löschen “.

    3. Wählen Sie im Dialogfeld „Benutzer löschen “ die Schaltfläche „ Löschen “.

    Synchronisieren von Videos

    Synchronisiere Videos unter dem aktuellen Account von Brightcove mit Kollective.

    Videoliste

    1. Wählen Sie in der linken Navigation „ Content Syndicator“ und dann „ Videos Syndicator“ aus.

      Sie sollten eine Liste aller Videos in Ihrem Video Cloud-Konto mit den neuesten hinzugefügten/aktualisierten Videos sehen.

    2. In der Videolist können Sie nach allen, synchronisierten oder nicht synchronisierten Videos filtern.

    3. Wählen Sie ein oder mehrere Videos in der Liste aus und klicken Sie auf die Schaltfläche Synchronisieren, um von Brightcove zu Kollective zu synchronisieren.

    Video Detail

    1. Wählen Sie ein beliebiges Video in der Liste aus, um die Videodetailseite anzuzeigen.

    2. Auf der Seite mit den Videodetails können Sie Folgendes tun:

      • Synchronisieren Sie das Video
      • Videostatus aktualisieren
      • Lösche das Video

    Synchronisieren von Live-Events

    Synchronisieren Sie Live-Events unter dem aktuellen Account von Brightcove mit Kollective.

    Videoliste

    1. Wählen Sie in der linken Navigation Content Syndicator und dann Live Events Syndicatoraus.

      Sie sollten eine Liste aller Live-Ereignisse in Ihrem Video Cloud-Konto mit den zuletzt hinzugefügten/aktualisierten Videos sehen.

    2. In der Live-Event-Liste können Sie nach All-, Live-, Standby- oder Abgeschlossenen Live-Events filtern.

    3. Wählen Sie ein oder mehrere Live-Events in der Liste aus und klicken Sie auf die Schaltfläche Synchronisieren, um von Brightcove zu Kollective zu synchronisieren.

    Live-Ereignis-Details

    1. Wählen Sie ein beliebiges Live-Event in der Liste aus, um die Detailseite anzuzeigen.

    2. Auf der Detailseite der Live-Veranstaltung können Sie Folgendes tun:

      • Synchronisieren Sie das Live-Event
      • Aktualisieren des Live-Ereignisstatus
      • Löschen Sie das Live-Event

    Syndikator-Protokolle prüfen

    Die Syndikator-Protokolle enthalten Erfolgs-, Fehler- und Warndetails zu Video- und Live-Ereignissynchronisierungen für Ihr Konto.

    1. Wählen Sie in der linken Navigation Syndicator-Logsaus.

      Sie sollten eine Liste der Synchronisationsprotokolle für das aktuelle Konto mit dem neuesten oben sehen.

    2. In der Liste der Synchronisationsprotokolle können Sie nach All, Error, Info oder Warningfiltern.

    Einbetten des Spielercodes

    Es ändert nichts an der Art und Weise, wie Sie den Player und das zugehörige Content-Asset in Ihre Webseite einbetten.

    1. Navigieren Sie in Video Cloud Studio von Brightcove zum Medienmodul.

    2. Wählen Sie das Video aus, das Sie verwenden möchten, und klicken Sie auf die Schaltfläche Veröffentlichen und Einbetten .

    3. Wählen Sie den Player aus, der für die Verwendung des Kollective-Plugins konfiguriert wurde.

    4. Kopieren Sie entweder den Standard- oder den Advanced-Code und fügen Sie ihn Ihrer Webseite hinzu.

      Standardmäßig läuft der Player-Token nach 6 Stunden nach der Synchronisierung von Inhalten ab. Nach Ablauf ist es nicht mehr möglich, Inhalte mehr abzuspielen. Um die Ablaufzeit zu ändern, wenden Sie sich an den Brightcove Support.
