Bereiten Sie die Installation vor

Dateien und Ordner extrahieren und in den Arbeitsordner kopieren

  1. Erstellen Sie einen Ordner [Magento_Root]/app/code/Brightcove/Upload
  2. Laden Sie die Archivdatei (Codedateien) herunter.
  3. Entpacken Sie die Dateien / Ordner
  4. Löschen / Verschieben des Inhalts der entpackten Dateien in das Verzeichnis in Schritt 1

Aktivieren Sie Erweiterungen und Plugins

Aktivieren Sie Folgendes:

  • php bin/magento module:enable Brigtcove_Upload
  • php bin/magento setup:upgrade
  • php bin/magento cache:clean
  • php bin/magento setup:static-content:deploy
  • php bin/magento cache:flush

Konfigurieren Sie die Anmeldeinformationen des Brightcove-Moduls

Um diesen Teil abzuschließen, müssen Sie Client-Anmeldeinformationen für die von der Erweiterung verwendeten APIs abrufen. Einzelheiten zum Erstellen von Anmeldeinformationen finden Sie unter Verwalten von API-Authentifizierungsanmeldeinformationen. Die Mindestberechtigungen, die Sie für die Anmeldeinformationen benötigen, sind nachstehend aufgeführt:

  1. Navigieren Sie zu Speichern -> Konfiguration -> Brightcove (linke Registerkarte) -> Brightcove-Videokonfiguration
  2. Geben Sie die Details für die Brightcove-Konfiguration ein:
    • Kunden ID:: REST-Client / Benutzer-ID
    • Kundengeheimnis:: REST Client Secret / Passwort.
    • API-Basis-URL::
    • ICH WÜRDE:: Brightcove Store ID
    • Kategorie Video Slot:: So stellen Sie die Anzahl der Video-Slots auf Kategoriespalte ein
    • Master Video Slot:: So stellen Sie die Anzahl der Video-Slots auf Master Column ein
    • Variant Video Slot: So stellen Sie die Anzahl der Video-Slots auf Variant Column ein
    • E-Mail-ID: So senden Sie E-Mails zum Massen-Upload-Status
  3. Sobald alle Details korrekt angegeben sind, dann speichern die Konfigurationsseite. Jetzt sollten Benutzer auf das benutzerdefinierte Brightcove-Modul zugreifen können.

Erweiterung deaktivieren

Wenn Sie die Brightcove-Erweiterung jemals deaktivieren müssen, verwenden Sie die folgenden Befehle:

  1. php bin/magento module:disable Brightcove_Upload
  2. php bin/magento setup:upgrade
  3. php bin/magento cache:clean
  4. php bin/magento setup:static-content:deploy
  5. php bin/magento cache:flush
