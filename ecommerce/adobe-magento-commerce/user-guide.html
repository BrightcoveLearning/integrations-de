    Benutzerhandbuch für Adobe Magento Commerce Extension

    Dieses Dokument enthält schrittweise Anweisungen zur Verwendung der Brightcove-Erweiterung für Adobe Magento Commerce.

    1. Melden Sie sich bei Magento Commerce an
    2. Beachten Sie den Link „Brightcove Module“ in der linken Navigation:
    3. Klicken Sie auf das Brightcove Icon, um das Brightcove Modul-Menü anzuzeigen:

    Hochladen von Videos

    1. Öffnen Sie im Brightcove Modul-Menü die Seite „ Videos hinzufügen“ .
    2. Wählen Sie die Store-Ansicht aus, zu der Sie das Video hinzufügen möchten.
    3. Fügen Sie den Videotitel hinzu (erforderlich).
    4. Suchen Sie nach der lokalen Videodatei (erforderlich).
    5. Suchen Sie nach der lokalen Thumbnail-Bilddatei (erforderlich).
    6. Suchen Sie nach der lokalen Poster-Bilddatei (optional).
    7. Geben Sie kommagetrennte Tags für das Video ein (optional).
    8. Geben Sie eine Beschreibung für das Video ein (optional).
    9. Klicken Sie auf Save.

    Hinweis: Die Store-Ansicht wird in den Video-Metadaten als Tag gespeichert. Dieses Tag wird für die Implementierung von Localization auf der Seite „Zuweisen“ verwendet.

    Zuweisen von Videos zu Kategorien und Produkten

    Videos können Kategorien und Produkten zugewiesen werden. In diesem Abschnitt werden wir uns im folgenden Abschnitt ansehen, wie Sie Videos Kategorien zuweisen und sie dann Produkten zuweisen.

    Zuweisen zu Kategorien

    1. Öffnen Sie im Brightcove Modul-Menü die Seite „ Videos zu Kategorien zuweisen“ .
    2. Die erste Seite enthält eine Liste von Kategorien, in denen Videos angezeigt werden, die ihnen derzeit zugewiesen sind, und es gibt Links zu Unassign oder Ändern vorhandener Videozuweisungen sowie Zuweisen von Videos zu einer Kategorie.

      Beachten Sie auch:

      • Der Bereichs-Selektor, mit dem Sie den Lokalisierungsfilter implementieren können
      • Wenn Sie den Mauszeiger über eine Video-ID bewegen, wird ein Miniaturbild des Videos angezeigt
    3. Klicken Sie neben einer zuzuweisenden Kategorie auf Zuweisen zum Öffnen eines Rasters von zuzuweisenden Videos:
    4. Wenn Sie das gesuchte Video nicht sehen, können Sie nach dem Videotitel suchen, um es zu finden.
    5. Verwenden Sie das Dropdown-Menü für die Lokalisierung, um die gewünschte Sprache zu erhalten.
    6. In der Video-Spalte :
      • Wählen Sie den Player für das ausgewählte Video
      • Verwenden Sie den Selektor, um den Video-Slot zu wählen
      Beachten Sie, dass die Anzahl der Slot-Werte aus den Konfigurationswerten generiert wird.
      .
    7. Wenn das Video dieser Kategorie bereits zugewiesen ist, wird ein orangefarbener Rahmen und eine Option „ Zuweisung aufheben “ angezeigt.

    Zuweisen zu Produkten

    1. Öffnen Sie im Brightcove Modul-Menü die Seite „ Videos zu Produkten zuweisen“ .
    2. Wie bei Kategorien können Sie Videos zuweisen oder bereits zugewiesene Videos aufheben oder ändern . Sie können auch mit der Maus über eine Video-ID fahren, um das Miniaturbild anzuzeigen.
    3. Klicken Sie auf die Links zuweisen , um ein Video zuzuweisen. Dies wird das Videoraster aufrufen:
    4. Wenn Sie das gesuchte Video nicht sehen, können Sie nach dem Videotitel suchen, um es zu finden.
    5. Verwenden Sie das Dropdown-Menü für die Lokalisierung, um die gewünschte Sprache zu erhalten.
    6. In der Video-Spalte :
      • Wählen Sie den Player für das ausgewählte Video
      • Verwenden Sie den Selektor, um den Video-Slot zu wählen
      Beachten Sie, dass die Anzahl der Slot-Werte aus den Konfigurationswerten generiert wird.
      .
    7. Wenn das Video diesem Produkt bereits zugewiesen ist, wird ein orangefarbener Rahmen und eine Option „ Zuweisung aufheben “ angezeigt.

    Generieren Sie Einbettungscode

    1. Rufen Sie im Menü des Brightcove Moduls die Seite „Einbettungscode generieren“ auf:
    2. Wählen Sie den Player, den Sie verwenden möchten, und ein Video aus, und klicken Sie dann auf Code abrufen , um den Basiscode (iframe) oder Advanced (In-Page) für den Player anzuzeigen und zu kopieren.
    3. Wir empfehlen, dass Sie den Basic-Einbettungscode verwenden, es sei denn, Sie passen den Player auf eine Weise an, die dies erfordert, und das In-Page-Skript.

    Brightcove Konfiguration

    Sie können die Brightcove-Konfiguration aktualisieren oder ergänzen.

    1. Gehe zu Gespeichert - > Brightcove Video Configuration:
    2. Die Client-ID, das Client-Geheimnis, die API-Base-URL (um Zugriffstoken zu erhalten) und die Shop-ID sollten bereits während der Installation ausgefüllt worden sein, aber Sie können diese Werte bei Bedarf aktualisieren.
    3. Sie können auch die Anzahl der Video-Slots für Kategorien, Master und Varianten festlegen.
    4. Wenn Sie die Option Massendaten-Import (im folgenden Abschnitt beschrieben) verwenden möchten, geben Sie eine durch Kommas getrennte Liste von E-Mail-Adressen ein, um Benachrichtigungen im Zusammenhang mit Massenimporten zu erhalten.

    Massen-Importdaten

    Es gibt verschiedene Arten von Daten, die Sie in großen Mengen importieren können, darunter eine Liste von Videos, die auf Brightcove hochgeladen werden sollen, Videokategoriezuweisungen und Videoproduktzuweisungen.

    1. Gehe zu System - > Import öffne die Seite Massenimport:
    2. Wählen Sie die Art von Import aus, die Sie durchführen möchten.
    3. Hier ist ein Beispiel für die Seite „Videos in Brightcove massenweise hochladen“:
    4. Wenn Sie keine CSV-Beispieldatei für den Import haben, können Sie diese über den Beispiel-Download-Link herunterladen.
    5. Wählen Sie das Importverhaltenaus:
      • Hinzufügen/Aktualisieren - Existierende Datensätze hinzufügen und ersetzen
      • Ersetzen - Ersetzen Sie die vorhandenen Datensätze
      • Löschen - Löschen Sie die bestehenden Datensätze
    6. Wählen Sie mit der Schaltfläche „ Durchsuchen “ Ihre CSV-Datei aus.
    7. Klicken Sie auf Daten prüfen , um sicherzustellen, dass die Daten gültig sind. (Wenn die CSV-Datei keine gültigen Datensätze enthält, wird die Importoption nicht angezeigt).
    8. Klicken Sie auf die Schaltfläche Importieren , um die Daten zu importieren. Sie sollten eine Erfolgsmeldung sehen.
