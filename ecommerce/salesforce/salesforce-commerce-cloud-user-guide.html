    Salesforce Commerce Cloud-Benutzerhandbuch

    Dieses Dokument enthält schrittweise Anweisungen zur Verwendung von Brightcove Salesforce Cloud Custom Cartridge zum Hochladen von Videodateien in die Brightcove Cloud/Server und zum Veröffentlichen dieser Dateien in der Storefront (Produktdetailseite, Kategorie-Landingpage, Produktlistenseite usw.) nach Bedarf.

    Mehrsprachiger Support

    Die Erweiterung bietet Unterstützung für mehrere Benutzeroberflächensprachen für Business Manager Interface. Hinweis: Diese Version der Erweiterung unterstützt Englisch und Französisch für UI-Sprache.

    Video hinzufügen

    Im Folgenden finden Sie die Schritte zum Hinzufügen von Videos.

    1. Navigieren Sie zu Merchant Tools - > Brightcove - Video > hinzufügen
    2. Wählen Sie Sprache aus dem Menü (die Standardeinstellung ist das für die Site ausgewählte Standardgebietsschema):
    3. Klicken Sie auf die Option „Auswählen “ neben dem Feld Video.
    4. Wählen Sie im angezeigten Dialogfeld die Videodatei aus:
    5. Klicken Sie auf die Option Auswählen neben dem Feld Miniaturansicht, und es wird ein ähnliches Popup-Dialogfeld angezeigt, in dem Sie die Miniaturbilddatei auswählen können.
    6. Klicken Sie auf die Option Auswählen neben dem Feld Poster, und es wird ein ähnliches Popup-Dialogfeld angezeigt, in dem Sie die Posterdatei auswählen können.
    7. Reichen Sie das Formular ein, um den Aufgessungsprozess einzuleiten:
    8. Validieren Sie die Erfolgsmeldung im Formular nach dem Absenden:
    9. Sie können überprüfen, ob das Video erfolgreich aufgenommen wurde, indem Sie sich bei Brightcove Studio anmelden und zum Medienmodul navigieren:
    10. Sie können auch auf das Video klicken, um die Details anzuzeigen und den Gebietsschema-Code zu überprüfen:

    Video einbetten

    Mit den Schritten zum Einbetten von Videos können Sie Code zum Rendern von Videos auf jeder ISML-Seite abrufen.

    1. Navigieren Sie zu Merchant Tools - > Brightcove - Video > einbetten.
    2. Wählen Sie das gewünschte Video und den Player aus und klicken Sie auf Code abrufen:
    3. Es erscheint ein Popup-Dialogfeld mit dem Code - wählen Sie den Codeblock aus und wird in die Zwischenablage kopiert:
    4. Fügen Sie den Code in Content Asset, Content Slot oder in eine beliebige ISML-Vorlage ein, um das Video zu rendern.

    Massen-Upload

    Der Massenupload ermöglicht das Aufnehmen von Videos in großer Menge, indem Sie CSV-Dateien hochladen. Der Job verarbeitet die CSV-Datei (en) und nimmt die Videos in das Brightcove-Konto auf. Eine Beispieldatei bulkUpload ist in der ZIP-Datei unter Meta-Verzeichnis angehängt.

    Beachten Sie videofile, dass die videoposter und auf öffentlich zugänglichen Pfaden verfügbar sein videothumbnail muss.
    1. Navigieren Sie zu Merchant Tools - > Brightcove - > Bulk Upload.
    2. Wählen Sie die CSV-Datei aus.
    3. Klicken Sie auf Senden:
    4. Navigieren Sie zu Administration - > Jobs.
    5. Wählen Sie Bulk-Upload-Job und klicken Sie auf Ausführen
    6. Überprüfen Sie den Status des Auftragsabschlusses.
    7. Überprüfen Sie das Videoobjekt im Brightcove-Konto.
    8. Die von verarbeitete Datei Massen-Upload Job wird nach verschoben massUpload_processed Ordner unter Temp Verzeichnis.

    Category

    1. Navigieren Sie zu Merchant Tools - > Brightcove - > Massenzuweisung.
    2. Wählen Sie Kategorieaus.
    3. Wählen Sie die CSV-Datei aus.
    4. Klicken Sie auf Senden:
    5. Die Datei wird im Temp-Verzeichnis unter dem Ordner BulkAssignCategory hochgeladen.
    6. Navigieren Sie zu Administration - > Jobs.
    7. Wählen Sie den Auftrag Bulk-Assign aus und klicken Sie auf Ausführen.
    8. Überprüfen Sie den Status des Auftragsabschlusses.
    9. Überprüfen Sie die zugewiesenen Videos.
    10. Die von Bulk-Assign -Job verarbeitete Datei wird in den Ordner bulkAssignCategory_Processed unter dem Temp-Verzeichnis verschoben.

    Produkt

    1. Navigieren Sie zu Merchant Tools - > Brightcove - > Massenzuweisung.
    2. Wählen Sie Produktaus.
    3. Wählen Sie die CSV-Datei aus.
    4. Klicken Sie auf Senden:
    5. Die Datei wird im Temp-Verzeichnis unter dem Ordner BulkAssignProduct hochgeladen.
    6. Navigieren Sie zu Administration - > Jobs.
    7. Wählen Sie den Auftrag Bulk-Assign aus und klicken Sie auf Ausführen.
    8. Überprüfen Sie den Status des Auftragsabschlusses.
    9. Überprüfen Sie die zugewiesenen Videos.
    10. Die von Bulk-Assign -Job verarbeitete Datei wird in den Ordner BulkAssignProduct_Processed unter dem Temp-Verzeichnis verschoben.
