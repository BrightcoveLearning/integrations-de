Unterstützung für mehrere Sprachen

Die Erweiterung bietet Unterstützung für mehrere UI-Sprachen für die Business Manager-Schnittstelle. Hinweis: Diese Version der Erweiterung unterstützt Englisch und Französisch für die UI-Sprache.

Video hinzufügen

Im Folgenden finden Sie die Schritte zum Hinzufügen von Videos.

  1. Navigieren Sie zu Merchant Tools -> Brightcove -> Video hinzufügen
  2. Wählen Sprache aus dem Menü (die Standardeinstellung ist das für die Site ausgewählte Standardgebietsschema):
  3. Drücke den wählen Option neben Videofeld.
  4. Wählen Sie im angezeigten Dialogfeld die Videodatei aus:
  5. Drücke den wählen Option neben dem Miniaturbildfeld, und ein ähnliches Popup-Dialogfeld wird angezeigt, in dem Sie die Miniaturbilddatei auswählen können.
  6. Drücke den wählen Option neben Posterfeld, und ein ähnliches Popup-Dialogfeld wird angezeigt, in dem Sie die Posterdatei auswählen können.
  7. Senden Sie das Formular, um den Aufnahmevorgang einzuleiten:
  8. Überprüfen Sie die Erfolgsmeldung auf dem Formular nach dem Absenden:
  9. Sie können überprüfen, ob das Video erfolgreich aufgenommen wurde, indem Sie sich bei Brightcove Studio anmelden und zum Medienmodul navigieren:
  10. Sie können auch auf das Video klicken, um die Details anzuzeigen und den Gebietsschema-Code zu überprüfen:

Video einbetten

Mit den Schritten zum Einbetten von Videos können Sie Code zum Rendern von Videos auf jeder ISML-Seite abrufen.

  1. Navigieren Sie zu Merchant Tools -> Brightcove -> Video einbetten.
  2. Wählen Sie das gewünschte Video und den gewünschten Player aus und klicken Sie auf Code abrufen::
  3. Ein Popup-Dialogfeld mit dem Code wird angezeigt. Wählen Sie den Codeblock aus, und er wird in die Zwischenablage kopiert:
  4. Fügen Sie den Code in das Inhaltselement, den Inhaltsbereich oder in eine beliebige ISML-Vorlage ein, um das Video zu rendern.

Massen-Upload

Beim Massen-Upload können Videos in großen Mengen aufgenommen werden, indem CSV-Dateien hochgeladen werden. Der Job verarbeitet die CSV-Datei (en) und nimmt die Videos in das Brightcove-Konto auf. Eine Beispiel-BulkUpload-Datei wird in der Zip-Datei unter dem Meta-Verzeichnis angehängt.

Beachten Sie, dass die videofile videoposter und videothumbnail muss auf öffentlich zugänglichen Pfaden verfügbar sein.
  1. Navigieren Sie zu Merchant Tools -> Brightcove -> Massen-Upload.
  2. Wählen Sie die CSV-Datei aus.
  3. Klicken Sie auf Senden:
  4. Navigieren Sie zu Administration -> Jobs.
  5. Wählen Massen-Upload Job und klicken Lauf.
  6. Überprüfen Sie den Status des Auftragsabschlusses.
  7. Überprüfen Sie das Videoobjekt im Brightcove-Konto.
  8. Die von verarbeitete Datei Massen-Upload Job wird verschoben nach massUpload_processed Ordner unter Temp Verzeichnis.

Der Job verarbeitet CSV-Dateien für Kategorie und Produkt unter derselben Ausführung. Der Job überprüft die Verzeichnisse auf Produkt und Kategorie und verarbeitet die Datei (en).

Kategorie

  1. Navigieren Sie zu Merchant Tools -> Brightcove -> Bulk Assign.
  2. Wählen Kategorie.
  3. Wählen Sie die CSV-Datei aus.
  4. Klicken einreichen::
  5. Die Datei wird in die hochgeladen Temp Verzeichnis unter dem BulkAssignCategory Mappe.
  6. Navigieren Sie zu Administration -> Jobs.
  7. Wähle aus Bulk-Assign Job und klicken Lauf.
  8. Überprüfen Sie den Status des Auftragsabschlusses.
  9. Überprüfen Sie die zugewiesenen Videos.
  10. Die vom Bulk-Assign-Job verarbeitete Datei wird in die Datei verschoben massAssignCategory_processed Ordner unter dem Temp-Verzeichnis.

Produkt

  1. Navigieren Sie zu Merchant Tools -> Brightcove -> Bulk Assign.
  2. Wählen Produkt.
  3. Wählen Sie die CSV-Datei aus.
  4. Klicken einreichen::
  5. Die Datei wird in die hochgeladen Temp Verzeichnis unter dem BulkAssignProduct Mappe.
  6. Navigieren Sie zu Administration -> Jobs.
  7. Wähle aus Bulk-Assign Job und klicken Lauf.
  8. Überprüfen Sie den Status des Auftragsabschlusses.
  9. Überprüfen Sie die zugewiesenen Videos.
  10. Die vom Bulk-Assign-Job verarbeitete Datei wird in die Datei verschoben massAssignProduct_processed Ordner unter dem Temp-Verzeichnis.
