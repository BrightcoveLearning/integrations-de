WordPress-Brightcove-Connector: Aufbau

Dieses Thema behandelt die Konfiguration des Brightcove-Connectors für Wordpress.

Einführung

Mit WordPress Brightcove Video Connector können Sie Brightcove Video Cloud-Videos und -Player in WordPress verwalten und Videos einfach in WordPress-Seiten einbetten.

Stellen Sie eine Verbindung zum Brightcove-Konto her

  1. Navigieren Sie zu den Brightcove-Einstellungen und klicken Sie auf Brightcove-Konto hinzufügen
  2. Der Quellenname kann beliebig sein, aber die Verwendung des gleichen Namens wie der Kontoname in Brightcove erleichtert die Identifizierung.
  3. Um die Konto-ID zu finden, melden Sie sich bei Brightcove Studio an und sehen Sie in der oberen linken Ecke Ihres Dashboards nach.
  4. Um die Client-ID und das Client-Geheimnis zu erhalten, müssen Sie eine Anwendung registrieren. Navigieren Sie zur API-Authentifizierung und klicken Sie auf Neue Anwendung hinzufügen.
  5. Geben Sie den Namen und die Kurzbeschreibung ein.
  6. Wählen Sie ein Konto. In der folgenden Abbildung sind alle erforderlichen Berechtigungen aufgeführt.
  7. Klicken Sie auf Speichern.
  8. Client ID und Client Secret werden angezeigt. Stellen Sie sicher, dass Sie Client Secret kopieren und an einem sicheren Ort speichern. Sie können nicht mehr darauf zugreifen.
  9. Klicken Sie auf "Anmeldeinformationen überprüfen". Bei Erfolg sollten Ihre Videos angezeigt werden.

Legen Sie das Aufnahmeprofil fest

Version 1.4.1 des Connectors und der vorherige Hardcode des Video Cloud-Aufnahmeprofils lauten "videocloud-default-v1".  Wenn für das Brightcove-Konto nur Dynamic Delivery oder CAE aktiviert ist, ist dieses Aufnahmeprofil nicht vorhanden und das Hochladen von Videos schlägt fehl.  Im Folgenden finden Sie einige Optionen, um dieses Verhalten im Connector zu umgehen.

 

  1. Erstellen Sie im Video Cloud-Konto ein neues Aufnahmeprofil mit dem im Connector verwendeten Standardprofilnamen.  Navigieren Sie zu Admin-> Ingest Profiles und erstellen Sie ein neues Profil oder duplizieren Sie ein vorhandenes Profil und nennen Sie es "videocloud-default-v1".  

 

  1. Oder ändern Sie den Namen des Aufnahmeprofils im Connector-Quellcode, indem Sie zunächst unter Admin-> Aufnahme im Video Cloud-Konto ermitteln, welches Aufnahmeprofil Sie verwenden möchten. Klicken Sie auf API-Namen anzeigen und notieren Sie ihn.
  1. Bearbeiten wp-content/plugins/brightcove-video-connect/includes/api/class-bc-cms-api.php und ändern Sie auf dem WordPress-Server das Profil vom Standardprofilnamen „videocloud-default-v1“' auf den gewünschten Profilnamen.  Zum Beispiel ändern
    public function video_upload( $video_id, $video_url, $profile = 'videocloud-default-v1' ) {
    zu
    public function video_upload( $video_id, $video_url, $profile = 'multi-platform-standard-static' ) {
  1. Oder ändern Sie den Connector, um das Standard-Aufnahmeprofil im Video Cloud-Konto zu verwenden. Bearbeiten Sie dazu wp-content/plugins/brightcove-video-connect/includes/api/class-bc-cms-api.php und kommentieren Sie die Zeile aus:
    $data = array( 'profile' => sanitize_text_field( $profile ) );
    mit /* */.

Sie haben nun die Konfiguration des WordPress-Brightcove-Connectors abgeschlossen. Weiter zu Verwenden des Connectors.
