Adobe-AEM-Brightcove-Connector: Installation

Dieses Thema behandelt die Installation des Brightcove-Connectors für Adobe Experience Manager (AEM).

Einführung

Mit dem Adobe-AEM-Brightcove Connector können Sie Brightcove Video Cloud-Videos und -Player in AEM verwalten und Videos einfach in AEM-Seiten einbetten.

Beachten Sie, dass dieses Handbuch AEM 6.3 verwendet. Der Anschluss arbeitet auch mit AEM 6.2, 6.4 und 6.5 und die Schritte sind ähnlich.

Einstieg

Wenn Sie noch keinen AEM-Entwickler haben Zum Beispiel können Sie einfach eine mit der JAR-Datei AEM Quickstart erstellen:

java  -Xmx1400M -jar AEM_6.3_Quickstart.jar

Wenn Sie diesen Schnellstart verwenden, wird Ihre AEM-Instanz standardmäßig unter folgender Adresse gehostet: http: // localhost: 4502. In den folgenden Abschnitten wird dieser Standardspeicherort angenommen. Wenn Sie also einen anderen Speicherort oder Port verwenden, müssen Sie die in den Schritten angezeigten URLs anpassen oder stattdessen das AEM-Navigationsmenü verwenden.

Den Stecker installieren

Die folgenden Schritte führen Sie durch die Installation des Connectors über die klassische Schnittstelle.

  • Gehen Sie zu https://github.com/brightcove/Adobe-AEM-Brightcove-Connector/releases/ und laden Sie den Konnektor für Ihre Version von AEM herunter (die brightcove-all... zip-Datei).
  • Starten Sie Ihre Entwicklungsinstanz von AEM.
  • Wechseln Sie nach der Anmeldung zum CRX Package Manager (http: // localhost: 4502 / crx / packmgr / index.jsp).
  • Klicken Sie auf die Schaltfläche Hochladen:
  • Navigieren Sie im Dialogfeld "Hochladen" zur Zip-Datei für den Connector und klicken Sie auf "OK", um sie hochzuladen.
  • Wenn das Paket in der Liste angezeigt wird, klicken Sie auf Installieren, um es zu installieren:
  • Gehen Sie zur Seite Konsolenpakete (http: // localhost: 4502 / system / console / bundles), um zu überprüfen, ob das Paket aktiv ist:

Wenn Sie diese Schritte ausgeführt haben, ist der Connector erfolgreich installiert, und Sie sollten mit dem fortfahren Aufbau.
